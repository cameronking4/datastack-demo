---
title: Instantiate template
description: Create a stack or pipeline from a template with variable substitution
---

```
POST /api/v1/templates/{templateId}/instantiate
```

Create a stack or pipeline from a template with variable substitution


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `templateId` | `string` | Yes | The templateId value. |

## Response `201`

Returns created.


## Example

```bash title="cURL"
curl -X POST "/api/v1/templates/example-tem/instantiate" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json"
```
