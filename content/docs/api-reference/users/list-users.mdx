`GET /api/v1/users`

List users with pagination, role, and team filters

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `role` | `string` | No | The role value. |
| `teamId` | `string` | No | The teamId value. |
| `status` | `string` | No | The status value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/users" \
  -H "Authorization: Bearer <accessToken>"
```
