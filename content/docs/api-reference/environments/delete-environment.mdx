---
title: Delete Environment
description: Delete an environment. Protected environments cannot be deleted.
---

```
DELETE /api/v1/environments/{environmentId}
```

Permanently delete a deployment environment. Protected environments must be unprotected first.

**Required scope:** `manage:environments`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `environmentId` | `string` | The environment identifier. |

## Response `204`

No content returned on success.

## Example

```bash title="cURL"
curl -X DELETE /api/v1/environments/env-003 \
  -H "Authorization: Bearer <accessToken>"
```
