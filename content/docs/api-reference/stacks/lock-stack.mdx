`POST /api/v1/stacks/{stackId}/lock`

Acquire an exclusive lock on a stack to prevent concurrent modifications

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stackId` | `string` | Yes | The stackId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `reason` | `string` | No | The reason value. |
| `ttlSeconds` | `integer` | No | The ttlSeconds value. |

## Response `200`

Success

## Response `409`

Stack is already locked

## Example

```bash title="cURL"
curl -X POST "/api/v1/stacks/{stackId}/lock" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "reason": "example-reason",
  "ttlSeconds": 10
}'
```
