---
title: Create or Update Notebook
description: Create a new notebook or overwrite an existing one.
---

```
PUT /api/v1/notebooks
```

Create a new notebook at the specified path, or overwrite an existing one.

**Required scope:** `manage:notebooks`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `path` | `string` | Yes | Workspace path for the notebook. |
| `language` | `string` | Yes | Language (`PYTHON`, `SQL`, `SCALA`, `R`). |
| `content` | `string` | No | Notebook source content. Defaults to empty. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `path` | `string` | Workspace path. |
| `language` | `string` | Notebook language. |
| `format` | `string` | Content format (`SOURCE`). |
| `content` | `string` | Notebook content. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `modifiedAt` | `string` | ISO 8601 modification timestamp. |

## Example

```bash title="cURL"
curl -X PUT /api/v1/notebooks \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "path": "/Workspace/ETL/new_pipeline",
    "language": "PYTHON",
    "content": "# New pipeline notebook"
  }'
```

```json title="Response"
{
  "path": "/Workspace/ETL/new_pipeline",
  "language": "PYTHON",
  "format": "SOURCE",
  "content": "# New pipeline notebook",
  "createdAt": "2024-06-10T12:00:00Z",
  "modifiedAt": "2024-06-10T12:00:00Z"
}
```
