---
title: Get Webhook
description: Retrieve details of a specific webhook.
---

```
GET /api/v1/webhooks/{webhookId}
```

Returns full details for a single webhook.

**Required scope:** `manage:webhooks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `webhookId` | `string` | The webhook identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Webhook identifier. |
| `url` | `string` | Delivery URL. |
| `events` | `string[]` | Subscribed events. |
| `active` | `boolean` | Active status. |
| `createdAt` | `string` | Creation timestamp. |
| `updatedAt` | `string` | Last update timestamp. |
| `lastTriggeredAt` | `string` | Last delivery timestamp. |
| `failureCount` | `number` | Consecutive failures. |

## Example

```bash title="cURL"
curl /api/v1/webhooks/wh-001 \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "id": "wh-001",
  "url": "https://example.com/hooks/datastack",
  "events": ["job.completed", "job.failed"],
  "active": true,
  "createdAt": "2024-03-01T09:00:00Z",
  "updatedAt": "2024-05-15T10:00:00Z",
  "lastTriggeredAt": "2024-06-10T02:15:33Z",
  "failureCount": 0
}
```
