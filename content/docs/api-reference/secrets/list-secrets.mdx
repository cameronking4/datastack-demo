---
title: List Secrets in Scope
description: List secret keys in a scope (values are redacted).
---

```
GET /api/v1/secrets/scopes/{scopeName}
```

Returns the secret keys in a scope. Secret values are never returned.

**Required scope:** `read:secrets`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `scopeName` | `string` | Scope name. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `scope` | `string` | Scope name. |
| `secrets` | `object[]` | Array of secret metadata. |
| `secrets[].key` | `string` | Secret key name. |
| `secrets[].lastUpdatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `secrets[].lastUpdatedBy` | `string` | Last updater identity. |
| `totalCount` | `number` | Total number of secrets in the scope. |

## Example

```bash title="cURL"
curl "/api/v1/secrets/scopes/aws-prod" \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "scope": "aws-prod",
  "secrets": [
    { "key": "access_key_id", "lastUpdatedAt": "2024-03-01T10:00:00Z", "lastUpdatedBy": "ada@datastack.dev" },
    { "key": "secret_access_key", "lastUpdatedAt": "2024-03-01T10:00:00Z", "lastUpdatedBy": "ada@datastack.dev" }
  ],
  "totalCount": 3
}
```
