`POST /api/v1/pipelines/{pipelineId}/start`

Start pipeline

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/pipelines/{pipelineId}/start" \
  -H "Authorization: Bearer <accessToken>"
```
