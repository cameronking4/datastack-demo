`GET /api/v1/clusters/{clusterId}`

Get cluster details including runtime metrics and configuration

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `clusterId` | `string` | Yes | The clusterId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/clusters/{clusterId}" \
  -H "Authorization: Bearer <accessToken>"
```
