---
title: Get Deployment
description: Get deployment details including per-layer status.
---

```
GET /api/v1/deployments/{deploymentId}
```

Returns detailed information about a deployment including step-by-step layer status.

**Required scope:** `read:deployments`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `deploymentId` | `string` | The deployment identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Deployment identifier. |
| `stackId` | `string` | Stack identifier. |
| `stackName` | `string` | Stack display name. |
| `version` | `number` | Stack version deployed. |
| `environment` | `string` | Target environment. |
| `status` | `string` | Deployment status. |
| `steps` | `object[]` | Per-layer deployment steps. |
| `steps[].layer` | `string` | Layer name. |
| `steps[].resourceType` | `string` | Resource type. |
| `steps[].resourceId` | `string` | Resource ID. |
| `steps[].action` | `string` | Action taken (`CREATE`, `UPDATE`, `NO_CHANGE`, `ROLLBACK`). |
| `steps[].status` | `string` | Step status (`SUCCEEDED`, `FAILED`, `SKIPPED`, `PENDING`). |
| `steps[].startedAt` | `string\|null` | Step start timestamp. |
| `steps[].completedAt` | `string\|null` | Step completion timestamp. |
| `steps[].logs` | `string` | Step log message. |
| `startedAt` | `string` | ISO 8601 start timestamp. |
| `completedAt` | `string\|null` | ISO 8601 completion timestamp. |
| `startedBy` | `string` | Initiator identifier. |
| `promotedFrom` | `string\|null` | Source deployment ID if promoted. |

curl /api/v1/deployments/deploy-003 \
  -H "Authorization: Bearer <accessToken>"
