`POST /api/v1/queries/{queryId}/execute`

Execute a saved SQL query with optional parameter overrides and result format

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `queryId` | `string` | Yes | The queryId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `parameters` | `object` | No | The parameters value. |
| `limit` | `integer` | No | The limit value. |
| `resultFormat` | `string` | No | The resultFormat value. |
| `timeout` | `integer` | No | The timeout value. |
| `catalog` | `string` | No | The catalog value. |
| `schema` | `string` | No | The schema value. |
| `warehouseOverride` | `string` | No | The warehouseOverride value. |

## Response `202`

Accepted

## Example

```bash title="cURL"
curl -X POST "/api/v1/queries/{queryId}/execute" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "parameters": {},
  "limit": 10,
  "resultFormat": "example-resultFormat",
  "timeout": 10,
  "catalog": "example-catalog",
  "schema": "example-schema",
  "warehouseOverride": "example-warehouseOverride"
}'
```
