---
title: Get Query
description: Retrieve a saved query by identifier.
---

```
GET /api/v1/queries/{queryId}
```

Returns a single saved query with full details.

**Required scope:** `read:queries`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `queryId` | `string` | Query identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Query identifier. |
| `name` | `string` | Display name. |
| `description` | `string` | Query description. |
| `warehouseId` | `string` | Target warehouse identifier. |
| `sql` | `string` | SQL statement. |
| `status` | `string` | Status (`SAVED`). |
| `tags` | `string[]` | Tags. |
| `lastExecutedAt` | `string` | ISO 8601 last execution timestamp. |
| `lastExecutionDurationMs` | `number` | Last execution duration in milliseconds. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `createdBy` | `string` | Creator identity. |

## Example

```bash title="cURL"
curl "/api/v1/queries/query-001" \
  -H "Authorization: Bearer <accessToken>"
```
