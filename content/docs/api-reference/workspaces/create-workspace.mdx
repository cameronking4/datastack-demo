`POST /api/v1/workspaces`

Create a new workspace with region, quotas, and initial configuration

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `displayName` | `string` | No | The displayName value. |
| `region` | `string` | Yes | The region value. |
| `tier` | `string` | No | The tier value. |
| `quotas` | `object` | No | The quotas value. |
| `settings` | `object` | No | The settings value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/workspaces" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "displayName": "example-displayName",
  "region": "example-region",
  "tier": "example-tier",
  "quotas": {},
  "settings": {}
}'
```
