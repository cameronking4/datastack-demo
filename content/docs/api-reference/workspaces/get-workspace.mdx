`GET /api/v1/workspaces/{workspaceId}`

Get workspace

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/workspaces/{workspaceId}" \
  -H "Authorization: Bearer <accessToken>"
```
