`DELETE /api/v1/connections/{connectionId}`

Delete a connection. Fails if any active pipelines or jobs reference it.

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `connectionId` | `string` | Yes | The connectionId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/connections/{connectionId}" \
  -H "Authorization: Bearer <accessToken>"
```
