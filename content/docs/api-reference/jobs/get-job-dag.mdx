```
POST /api/v1/jobs/dag
```

Get job DAG


## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `jobIds` | `string[]` | No | The jobIds value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X POST "/api/v1/jobs/dag" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "jobIds": []
}'
```
