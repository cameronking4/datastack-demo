`GET /api/v1/query-history`

List query history

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `warehouseId` | `string` | No | The warehouseId value. |
| `userId` | `string` | No | The userId value. |
| `status` | `string` | No | The status value. |
| `limit` | `integer` | No | The limit value. |
| `offset` | `integer` | No | The offset value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/query-history" \
  -H "Authorization: Bearer <accessToken>"
```
