`GET /api/v1/environments/{environmentId}`

Get environment details including promotion rules, variables, and deployment count

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `environmentId` | `string` | Yes | The environmentId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/environments/{environmentId}" \
  -H "Authorization: Bearer <accessToken>"
```
