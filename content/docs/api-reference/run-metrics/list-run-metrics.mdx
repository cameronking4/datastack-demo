`GET /api/v1/run-metrics`

List run metrics

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | No | The jobId value. |
| `runId` | `string` | No | The runId value. |
| `from` | `string` | No | The from value. |
| `to` | `string` | No | The to value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/run-metrics" \
  -H "Authorization: Bearer <accessToken>"
```
