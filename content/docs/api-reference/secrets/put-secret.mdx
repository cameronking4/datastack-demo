---
title: Put Secret
description: Create or update a secret in a scope.
---

```
PUT /api/v1/secrets
```

Create or update a secret value within a scope. If the secret already exists, its value is overwritten.

**Required scope:** `manage:secrets`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `scope` | `string` | Yes | Secret scope name. |
| `key` | `string` | Yes | Secret key name. |
| `value` | `string` | Yes | Secret value (stored encrypted). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `scope` | `string` | Secret scope name. |
| `key` | `string` | Secret key name. |
| `lastUpdatedAt` | `string` | ISO 8601 update timestamp. |
| `lastUpdatedBy` | `string` | Updater identifier. |

## Example

```bash title="cURL"
curl -X PUT /api/v1/secrets \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "scope": "aws-prod",
    "key": "access_key_id",
    "value": "AKIA..."
  }'
```

```json title="Response"
{
  "scope": "aws-prod",
  "key": "access_key_id",
  "lastUpdatedAt": "2024-06-10T10:00:00Z",
  "lastUpdatedBy": "api"
}
```
