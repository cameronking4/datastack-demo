`POST /api/v1/catalogs/{catalogName}/schemas`

Create schema

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalogName` | `string` | Yes | The catalogName value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `comment` | `string` | No | The comment value. |
| `properties` | `object` | No | The properties value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/catalogs/{catalogName}/schemas" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "comment": "example-comment",
  "properties": {}
}'
```
