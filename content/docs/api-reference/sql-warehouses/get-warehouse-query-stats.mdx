`GET /api/v1/sql/warehouses/{warehouseId}/query-stats`

Get warehouse query stats

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `warehouseId` | `string` | Yes | The warehouseId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/sql/warehouses/{warehouseId}/query-stats" \
  -H "Authorization: Bearer <accessToken>"
```
