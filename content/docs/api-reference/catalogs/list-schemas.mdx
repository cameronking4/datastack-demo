`GET /api/v1/catalogs/{catalogName}/schemas`

List schemas in a catalog

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalogName` | `string` | Yes | The catalogName value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/catalogs/{catalogName}/schemas" \
  -H "Authorization: Bearer <accessToken>"
```
