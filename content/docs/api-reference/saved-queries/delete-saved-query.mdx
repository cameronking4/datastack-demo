```
DELETE /api/v1/saved-queries/{savedQueryId}
```

Delete saved query


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `savedQueryId` | `string` | Yes | The savedQueryId value. |

## Response `204`

Returns no content.


## Example

```bash title="cURL"
curl -X DELETE "/api/v1/saved-queries/example-sav" \
  -H "Authorization: Bearer <accessToken>"
```
