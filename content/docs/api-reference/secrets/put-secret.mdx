---
title: Put Secret
description: Create or update a secret in a scope.
---

```
PUT /api/v1/secrets
```

Creates a new secret or updates an existing one within a scope.

**Required scope:** `manage:secrets`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `scope` | `string` | Yes | Target scope name. |
| `key` | `string` | Yes | Secret key name. |
| `value` | `string` | Yes | Secret value. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `scope` | `string` | Scope name. |
| `key` | `string` | Secret key. |
| `lastUpdatedAt` | `string` | ISO 8601 update timestamp. |
| `lastUpdatedBy` | `string` | Updater identity. |

## Example

```bash title="cURL"
curl -X PUT "/api/v1/secrets" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "scope": "aws-prod", "key": "access_key_id", "value": "AKIAIOSFODNN7EXAMPLE" }'
```

```json title="Response"
{
  "scope": "aws-prod",
  "key": "access_key_id",
  "lastUpdatedAt": "2024-06-10T10:00:00Z",
  "lastUpdatedBy": "api"
}
```
