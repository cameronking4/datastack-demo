`GET /api/v1/pipelines/{pipelineId}/runs`

List execution history for a pipeline with duration, status, and error details

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `status` | `string` | No | The status value. |
| `limit` | `integer` | No | The limit value. |
| `offset` | `integer` | No | The offset value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/pipelines/{pipelineId}/runs" \
  -H "Authorization: Bearer <accessToken>"
```
