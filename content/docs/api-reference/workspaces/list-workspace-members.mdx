`GET /api/v1/workspaces/{workspaceId}/members`

List members of a workspace with roles and last activity

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `role` | `string` | No | The role value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/workspaces/{workspaceId}/members" \
  -H "Authorization: Bearer <accessToken>"
```
