---
title: Update resource tags
description: Set or update tags on a catalog resource
---

```
PUT /api/v1/catalog/{resourceType}/{resourceId}/tags
```

Set or update tags on a catalog resource


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `resourceType` | `string` | Yes | The resourceType value. |
| `resourceId` | `string` | Yes | The resourceId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `tags` | `string[]` | No | The tags value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X PUT "/api/v1/catalog/example-res/example-res/tags" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "tags": []
}'
```
