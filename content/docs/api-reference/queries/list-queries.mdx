`GET /api/v1/queries`

List saved SQL queries with optional filters

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `warehouseId` | `string` | No | The warehouseId value. |
| `ownerId` | `string` | No | The ownerId value. |
| `tag` | `string` | No | The tag value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/queries" \
  -H "Authorization: Bearer <accessToken>"
```
