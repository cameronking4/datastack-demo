`GET /api/v1/workspaces/{workspaceId}/templates`

List reusable templates available in the workspace

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `category` | `string` | No | The category value. |
| `page` | `integer` | No | The page value. |
| `pageSize` | `integer` | No | The pageSize value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/workspaces/{workspaceId}/templates" \
  -H "Authorization: Bearer <accessToken>"
```
