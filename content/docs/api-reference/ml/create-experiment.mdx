```
POST /api/v1/ml/experiments
```

Create experiment


## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `modelId` | `string` | No | The modelId value. |
| `parameters` | `object` | No | The parameters value. |

## Response `201`

Returns created.


## Example

```bash title="cURL"
curl -X POST "/api/v1/ml/experiments" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "modelId": "example-modelId",
  "parameters": {}
}'
```
