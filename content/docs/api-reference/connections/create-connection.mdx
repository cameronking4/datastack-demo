`POST /api/v1/connections`

Create a new data source or sink connection with authentication

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `type` | `string` | Yes | The type value. |
| `host` | `string` | Yes | The host value. |
| `port` | `integer` | No | The port value. |
| `workspaceId` | `string` | Yes | The workspaceId value. |
| `authMethod` | `string` | Yes | The authMethod value. |
| `credentialRef` | `string` | No | The credentialRef value. |
| `healthCheck` | `object` | No | The healthCheck value. |
| `properties` | `object` | No | The properties value. |
| `tags` | `object` | No | The tags value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/connections" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "type": "example-type",
  "host": "example-host",
  "port": 10,
  "workspaceId": "example-workspaceId",
  "authMethod": "example-authMethod",
  "credentialRef": "example-credentialRef",
  "healthCheck": {},
  "properties": {},
  "tags": {}
}'
```
