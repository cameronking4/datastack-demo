`POST /api/v1/jobs/{jobId}/runs`

Trigger job run

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `notebookParams` | `object` | No | The notebookParams value. |
| `idempotencyToken` | `string` | No | The idempotencyToken value. |
| `queueDuration` | `string` | No | The queueDuration value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/jobs/{jobId}/runs" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "notebookParams": {},
  "idempotencyToken": "example-idempotencyToken",
  "queueDuration": "example-queueDuration"
}'
```
