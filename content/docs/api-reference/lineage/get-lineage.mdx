---
title: Get Data Lineage Graph
description: Retrieve the lineage graph for a resource showing upstream and downstream dependencies.
---

```
GET /api/v1/lineage
```

Retrieve the lineage graph for a resource showing upstream and downstream dependencies, column-level lineage, and transformation metadata.

**Required scope:** `read:lineage`

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `resourceId` | `string` | Yes | Resource identifier to trace lineage for. |
| `resourceType` | `string` | Yes | Resource type (`table`, `pipeline`, `dashboard`). |
| `depth` | `integer` | No | Maximum traversal depth (default: `3`). |
| `direction` | `string` | No | Traversal direction (`upstream`, `downstream`, `both`). Default: `both`. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `resourceId` | `string` | Root resource identifier. |
| `resourceType` | `string` | Root resource type. |
| `nodes` | `object[]` | Graph nodes. |
| `nodes[].id` | `string` | Node identifier. |
| `nodes[].type` | `string` | Node resource type. |
| `nodes[].name` | `string` | Node display name. |
| `edges` | `object[]` | Graph edges. |
| `edges[].source` | `string` | Source node identifier. |
| `edges[].target` | `string` | Target node identifier. |
| `edges[].columns` | `object[]` | Column-level lineage mappings. |

## Example

```bash title="cURL"
curl "/api/v1/lineage?resourceId=table-001&resourceType=table&depth=2&direction=upstream" \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "resourceId": "table-001",
  "resourceType": "table",
  "nodes": [
    { "id": "table-001", "type": "table", "name": "gold.customers" },
    { "id": "pipe-001", "type": "pipeline", "name": "Customer ETL" },
    { "id": "table-002", "type": "table", "name": "bronze.raw_customers" }
  ],
  "edges": [
    {
      "source": "table-002",
      "target": "pipe-001",
      "columns": [{ "source": "raw_id", "target": "customer_id" }]
    },
    {
      "source": "pipe-001",
      "target": "table-001",
      "columns": [{ "source": "customer_id", "target": "id" }]
    }
  ]
}
```
