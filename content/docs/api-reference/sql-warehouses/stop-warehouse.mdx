---
title: Stop SQL Warehouse
description: Stop a running SQL warehouse.
---

```
POST /api/v1/sql/warehouses/{warehouseId}/stop
```

Stop a running SQL warehouse. The warehouse transitions to `STOPPING` state.

**Required scope:** `manage:sql`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `warehouseId` | `string` | The warehouse identifier. |

## Response `200`

Returns the warehouse object with state `STOPPING`.

## Example

```bash title="cURL"
curl -X POST /api/v1/sql/warehouses/wh-001/stop \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "id": "wh-001",
  "name": "BI Warehouse",
  "clusterSize": "Small",
  "state": "STOPPING",
  "jdbcUrl": "jdbc:datastack:sql://wh-001.us-west-2.datastack.cloud:443",
  "odbcUrl": "Driver={DataStack};Server=wh-001.us-west-2.datastack.cloud;Port=443",
  "createdAt": "2024-01-20T10:00:00Z"
}
```
