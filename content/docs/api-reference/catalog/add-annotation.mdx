---
title: Add annotation
description: Add an annotation to a catalog resource
---

`POST /api/v1/catalog/{resourceType}/{resourceId}/annotations`

Add an annotation to a catalog resource

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `resourceType` | `string` | Yes | The resourceType value. |
| `resourceId` | `string` | Yes | The resourceId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `content` | `string` | No | The content value. |
| `type` | `string` | No | The type value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/catalog/{resourceType}/{resourceId}/annotations" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "content": "example-content",
  "type": "example-type"
}'
```
