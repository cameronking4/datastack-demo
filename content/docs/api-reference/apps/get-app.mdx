```
GET /api/v1/apps/{appId}
```

Get app


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `appId` | `string` | Yes | The appId value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X GET "/api/v1/apps/example-app" \
  -H "Authorization: Bearer <accessToken>"
```
