## Sample Questions & Answers

### **Q1: How do I authenticate with the DataStack API?**

**A:**  
Authentication can be performed using either an API key or a Bearer token (JWT).  
- To use an API key, include the header:  
  `x-api-key: YOUR_API_KEY`  
- To use Bearer authentication, include the header:  
  `Authorization: Bearer YOUR_JWT_TOKEN`  

Please refer to our [Authentication Guide](/docs/auth/authentication) for full details.

### **Q3: How can I revoke an API key?**

**A:**  
You can revoke an API key via the API with the following request:

```
DELETE /api/v1/auth/api-keys/{keyId}
```

Replace `{keyId}` with the ID of the API key you wish to revoke.
A successful revocation returns a `200` status.
