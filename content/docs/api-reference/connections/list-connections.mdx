---
title: List Connections
description: Retrieve a paginated list of external data source and sink connections.
---

```
GET /api/v1/connections
```

Returns a paginated list of connections to external data sources and sinks.

**Required scope:** `read:connections`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number (min 1). |
| `pageSize` | `number` | `25` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `connections` | `object[]` | Array of connection objects. |
| `connections[].id` | `string` | Connection identifier. |
| `connections[].name` | `string` | Display name. |
| `connections[].type` | `string` | Connection type (`S3`, `POSTGRESQL`, `KAFKA`, `SNOWFLAKE`, etc.). |
| `connections[].host` | `string` | Host URI or address. |
| `connections[].status` | `string` | Current status (`ACTIVE`, `INACTIVE`, `PENDING_VALIDATION`). |
| `connections[].workspaceId` | `string` | Workspace identifier. |
| `connections[].secretScope` | `string` | Secret scope for credentials. |
| `connections[].properties` | `object` | Connection-specific properties. |
| `connections[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `connections[].updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `connections[].createdBy` | `string` | Creator identifier. |
| `connections[].lastTestedAt` | `string` | Last connectivity test timestamp. |
| `connections[].lastTestStatus` | `string` | Last test result (`SUCCESS`, `FAILURE`). |
| `totalCount` | `number` | Total number of connections. |
| `page` | `number` | Current page. |
| `pageSize` | `number` | Page size. |

## Example

```bash title="cURL"
curl "/api/v1/connections?page=1&pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```
