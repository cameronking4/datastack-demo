---
title: List secrets in scope
description: List secret keys in a scope (values are redacted)
---

`GET /api/v1/secrets/scopes/{scopeName}`

List secret keys in a scope (values are redacted)

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `scopeName` | `string` | Yes | The scopeName value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/secrets/scopes/{scopeName}" \
  -H "Authorization: Bearer <accessToken>"
```
