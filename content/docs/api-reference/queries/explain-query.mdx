`POST /api/v1/queries/{queryId}/explain`

Get execution plan and suggestions for a query

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `queryId` | `string` | Yes | The queryId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `format` | `string` | No | The format value. |
| `verbose` | `boolean` | No | The verbose value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/queries/{queryId}/explain" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "format": "example-format",
  "verbose": true
}'
```
