`PATCH /api/v1/queries/{queryId}`

Update query text, parameters, or metadata

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `queryId` | `string` | Yes | The queryId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `description` | `string` | No | The description value. |
| `queryText` | `string` | No | The queryText value. |
| `warehouseId` | `string` | No | The warehouseId value. |
| `catalog` | `string` | No | The catalog value. |
| `schema` | `string` | No | The schema value. |
| `resultFormat` | `string` | No | The resultFormat value. |
| `timeout` | `integer` | No | The timeout value. |
| `parameters` | `array<object>` | No | The parameters value. |
| `tags` | `array<string>` | No | The tags value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/queries/{queryId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "description": "example-description",
  "queryText": "example-queryText",
  "warehouseId": "example-warehouseId",
  "catalog": "example-catalog",
  "schema": "example-schema",
  "resultFormat": "example-resultFormat",
  "timeout": 10,
  "parameters": [],
  "tags": []
}'
```
