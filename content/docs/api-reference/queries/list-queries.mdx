---
title: List Queries
description: List saved SQL queries.
---

```
GET /api/v1/queries
```

Returns a paginated list of saved SQL queries.

**Required scope:** `read:queries`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number (min 1). |
| `pageSize` | `number` | `25` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `queries` | `object[]` | Array of query objects. |
| `queries[].id` | `string` | Query identifier. |
| `queries[].name` | `string` | Display name. |
| `queries[].description` | `string` | Query description. |
| `queries[].warehouseId` | `string` | Target warehouse identifier. |
| `queries[].sql` | `string` | SQL statement. |
| `queries[].status` | `string` | Status (`SAVED`). |
| `queries[].lastExecutedAt` | `string` | ISO 8601 last execution timestamp. |
| `queries[].lastExecutionDurationMs` | `number` | Last execution duration in milliseconds. |
| `queries[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `queries[].updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `queries[].createdBy` | `string` | Creator identity. |
| `totalCount` | `number` | Total number of queries. |
| `page` | `number` | Current page number. |
| `pageSize` | `number` | Results per page. |

## Example

```bash title="cURL"
curl "/api/v1/queries?page=1&pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```
