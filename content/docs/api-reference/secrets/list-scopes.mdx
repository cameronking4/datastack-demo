---
title: List Secret Scopes
description: List all secret scopes in the workspace.
---

```
GET /api/v1/secrets/scopes
```

Returns a list of secret scopes.

**Required scope:** `read:secrets`

## Response `200`

| Field | Type | Description |
|---|---|---|
| `scopes` | `object[]` | Array of scope objects. |
| `scopes[].name` | `string` | Scope name. |
| `scopes[].backendType` | `string` | Storage backend (`DATASTACK`). |
| `scopes[].secretsCount` | `number` | Number of secrets in the scope. |
| `scopes[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `scopes[].createdBy` | `string` | Creator identity. |
| `totalCount` | `number` | Total number of scopes. |

## Example

```bash title="cURL"
curl "/api/v1/secrets/scopes" \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "scopes": [
    {
      "name": "aws-prod",
      "backendType": "DATASTACK",
      "secretsCount": 3,
      "createdAt": "2024-01-15T10:00:00Z",
      "createdBy": "ada@datastack.dev"
    }
  ],
  "totalCount": 3
}
```
