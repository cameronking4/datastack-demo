`GET /api/v1/data-quality/reports/{tableId}`

Get data quality report and score for a table

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `tableId` | `string` | Yes | The tableId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/data-quality/reports/{tableId}" \
  -H "Authorization: Bearer <accessToken>"
```
