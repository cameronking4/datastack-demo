`PATCH /api/v1/users/{id}`

Update user profile, role, teams, or permissions

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `id` | `string` | Yes | The id value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `displayName` | `string` | No | The displayName value. |
| `email` | `string` | No | The email value. |
| `role` | `string` | No | The role value. |
| `department` | `string` | No | The department value. |
| `teamIds` | `array<string>` | No | The teamIds value. |
| `permissions` | `array<string>` | No | The permissions value. |
| `mfaEnabled` | `boolean` | No | The mfaEnabled value. |
| `status` | `string` | No | The status value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/users/{id}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "displayName": "example-displayName",
  "email": "example-email",
  "role": "example-role",
  "department": "example-department",
  "teamIds": [],
  "permissions": [],
  "mfaEnabled": true,
  "status": "example-status"
}'
```
