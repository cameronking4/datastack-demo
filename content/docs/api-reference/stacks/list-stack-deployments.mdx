---
title: List Stack Deployments
description: List deployment history for a stack.
---

```
GET /api/v1/stacks/{stackId}/deployments
```

Returns the deployment history for a specific stack, ordered by most recent first.

**Required scope:** `read:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | The stack identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `deployments` | `object[]` | Array of deployment objects. |
| `deployments[].id` | `string` | Deployment identifier. |
| `deployments[].stackId` | `string` | Stack identifier. |
| `deployments[].version` | `number` | Stack version that was deployed. |
| `deployments[].environment` | `string` | Target environment. |
| `deployments[].status` | `string` | Deployment status (`SUCCEEDED`, `FAILED`, `IN_PROGRESS`). |
| `deployments[].layersDeployed` | `number` | Number of layers successfully deployed. |
| `deployments[].layersFailed` | `number` | Number of layers that failed. |
| `deployments[].startedAt` | `string` | ISO 8601 start timestamp. |
| `deployments[].completedAt` | `string` | ISO 8601 completion timestamp. |
| `deployments[].startedBy` | `string` | Initiator identifier. |
| `totalCount` | `number` | Total number of deployments. |

## Example

```bash title="cURL"
curl /api/v1/stacks/stack-001/deployments \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "deployments": [
    {
      "id": "deploy-003",
      "stackId": "stack-001",
      "version": 3,
      "environment": "PRODUCTION",
      "status": "SUCCEEDED",
      "layersDeployed": 4,
      "layersFailed": 0,
      "startedAt": "2024-06-10T08:00:00Z",
      "completedAt": "2024-06-10T08:05:33Z",
      "startedBy": "ada@datastack.dev"
    }
  ],
  "totalCount": 3
}
```
