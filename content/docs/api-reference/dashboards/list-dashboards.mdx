`GET /api/v1/dashboards`

List saved dashboards (widget layouts)

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `page` | `integer` | No | The page value. |
| `pageSize` | `integer` | No | The pageSize value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/dashboards" \
  -H "Authorization: Bearer <accessToken>"
```
