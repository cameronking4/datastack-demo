`GET /api/v1/workspaces/{workspaceId}/roles`

List user-role assignments in the workspace

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `page` | `integer` | No | The page value. |
| `pageSize` | `integer` | No | The pageSize value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/workspaces/{workspaceId}/roles" \
  -H "Authorization: Bearer <accessToken>"
```
