---
title: Delete Alert
description: Delete a monitoring alert.
---

```
DELETE /api/v1/alerts/{alertId}
```

Permanently delete a monitoring alert.

**Required scope:** `manage:alerts`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `alertId` | `string` | The alert identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `message` | `string` | Confirmation message. |

## Example

```bash title="cURL"
curl -X DELETE /api/v1/alerts/alert-001 \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "message": "Alert alert-001 deleted."
}
```
