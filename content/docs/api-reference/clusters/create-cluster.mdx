title: Create cluster
description: Create a new compute cluster with autoscaling, spot instances, and custom configurations
---

`POST /api/v1/clusters`

Create a new compute cluster with autoscaling, spot instances, and custom configurations

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `workspaceId` | `string` | Yes | The workspaceId value. |
| `runtimeVersion` | `string` | Yes | The runtimeVersion value. |
| `nodeType` | `string` | Yes | The nodeType value. |
| `driverNodeType` | `string` | No | The driverNodeType value. |
| `numWorkers` | `integer` | No | The numWorkers value. |
| `enableAutoscaling` | `boolean` | No | The enableAutoscaling value. |
| `minWorkers` | `integer` | No | The minWorkers value. |
| `maxWorkers` | `integer` | No | The maxWorkers value. |
| `autoTerminationMinutes` | `integer` | No | The autoTerminationMinutes value. |
| `spotInstances` | `boolean` | No | The spotInstances value. |
| `spotFallbackToOnDemand` | `boolean` | No | The spotFallbackToOnDemand value. |
| `instancePoolId` | `string` | No | The instancePoolId value. |
| `clusterPolicyId` | `string` | No | The clusterPolicyId value. |
| `runtimeEngine` | `string` | No | The runtimeEngine value. |
| `initScripts` | `array<object>` | No | The initScripts value. |
| `customSparkConf` | `object` | No | The customSparkConf value. |
| `sshPublicKeys` | `array<string>` | No | The sshPublicKeys value. |
| `tags` | `object` | No | The tags value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/clusters" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "workspaceId": "example-workspaceId",
  "runtimeVersion": "example-runtimeVersion",
  "nodeType": "example-nodeType",
  "driverNodeType": "example-driverNodeType",
  "numWorkers": 10,
  "enableAutoscaling": true,
  "minWorkers": 10,
  "maxWorkers": 10,
  "autoTerminationMinutes": 10,
  "spotInstances": true,
  "spotFallbackToOnDemand": true,
  "instancePoolId": "example-instancePoolId",
  "clusterPolicyId": "example-clusterPolicyId",
  "runtimeEngine": "example-runtimeEngine",
  "initScripts": [],
  "customSparkConf": {},
  "sshPublicKeys": [],
  "tags": {}
}'
```
