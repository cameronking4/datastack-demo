`DELETE /api/v1/dashboards/{dashboardId}`

Delete dashboard

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `dashboardId` | `string` | Yes | The dashboardId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/dashboards/{dashboardId}" \
  -H "Authorization: Bearer <accessToken>"
```
