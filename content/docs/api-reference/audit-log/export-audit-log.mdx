`POST /api/v1/audit-log/export`

Request an audit log export (async job; download URL provided when ready)

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `since` | `string` | No | The since value. |
| `until` | `string` | No | The until value. |
| `format` | `string` | No | The format value. |
| `resourceTypes` | `array<string>` | No | The resourceTypes value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/audit-log/export" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "since": "example-since",
  "until": "example-until",
  "format": "example-format",
  "resourceTypes": []
}'
```
