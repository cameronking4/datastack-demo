`GET /api/v1/streams/{streamId}/metrics`

Get streaming job throughput and latency metrics

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `streamId` | `string` | Yes | The streamId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/streams/{streamId}/metrics" \
  -H "Authorization: Bearer <accessToken>"
```
