---
title: List Workspaces
description: Retrieve a list of workspaces the caller has access to.
---

```
GET /api/v1/workspaces
```

Returns a list of workspaces accessible to the authenticated user.

**Required scope:** `read:workspaces`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `pageSize` | `number` | `25` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `workspaces` | `object[]` | Array of workspace objects. |
| `workspaces[].id` | `string` | Workspace identifier. |
| `workspaces[].name` | `string` | Display name. |
| `workspaces[].region` | `string` | Deployment region. |
| `workspaces[].deploymentUrl` | `string` | Workspace URL. |
| `workspaces[].status` | `string` | Workspace status (`ACTIVE`). |
| `workspaces[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `totalCount` | `number` | Total number of workspaces. |
| `nextPageToken` | `string\|null` | Token for next page, or `null`. |

## Example

```bash title="cURL"
curl "/api/v1/workspaces?pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```
