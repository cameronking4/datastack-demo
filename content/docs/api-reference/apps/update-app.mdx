`PATCH /api/v1/apps/{appId}`

Update app

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `appId` | `string` | Yes | The appId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/apps/{appId}" \
  -H "Authorization: Bearer <accessToken>"
```
