---

`DELETE /api/v1/environments/{environmentId}`

Delete an environment. Protected environments cannot be deleted.

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `environmentId` | `string` | Yes | The environmentId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/environments/{environmentId}" \
  -H "Authorization: Bearer <accessToken>"
```
