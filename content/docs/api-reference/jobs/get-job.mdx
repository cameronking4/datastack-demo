`GET /api/v1/jobs/{jobId}`

Get job configuration, schedule, and recent run history

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/jobs/{jobId}" \
  -H "Authorization: Bearer <accessToken>"
```
