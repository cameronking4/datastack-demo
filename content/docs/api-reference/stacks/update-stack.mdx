---
title: Update Stack
description: Update a stack configuration.
---

```
PATCH /api/v1/stacks/{stackId}
```

Updates one or more fields of an existing stack.

**Required scope:** `manage:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | Stack identifier. |

## Request Body

All fields are optional. Only provided fields are updated.

| Field | Type | Description |
|---|---|---|
| `name` | `string` | New display name. |
| `description` | `string` | New description. |
| `version` | `string` | Stack version label. |
| `layers` | `object[]` | Replacement layer list. |
| `layers[].name` | `string` | Layer name. |
| `layers[].resourceType` | `string` | Resource type. |
| `layers[].resourceId` | `string` | Referenced resource identifier. |
| `layers[].order` | `integer` | Execution order. |
| `layers[].config` | `object` | Layer-specific configuration. |
| `layers[].dependsOn` | `string[]` | Layer dependencies. |
| `connections` | `string[]` | Replacement connection list. |
| `tags` | `object` | Replacement key-value tags. |
| `autoDeployOnPush` | `boolean` | Enable auto-deploy on git push. |
| `gitRepository` | `string` | Git repository URL. |
| `gitBranch` | `string` | Git branch name. |

## Response `200`

Returns the updated stack object.

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/stacks/stack-001" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Analytics Platform v2", "gitRepository": "https://github.com/acme/infra" }'
```
