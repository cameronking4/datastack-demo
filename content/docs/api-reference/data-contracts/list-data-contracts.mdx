---
title: List Data Contracts
description: Retrieve a paginated list of data contracts with optional filters.
---

```
GET /api/v1/data-contracts
```

Returns a paginated list of data contracts. You can filter by status, owner team, or table name.

**Required scope:** `read:data-contracts`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number. |
| `pageSize` | `number` | `25` | Results per page (1-100). |
| `status` | `string` | — | Filter by status (`ACTIVE`, `DRAFT`, `VIOLATED`, `EXPIRED`, `ARCHIVED`). |
| `owner` | `string` | — | Filter by producer or consumer team name. |
| `table` | `string` | — | Filter by table name (partial match). |
| `sortBy` | `string` | — | Sort field (`name`, `createdAt`, `updatedAt`, `status`). |
| `sortOrder` | `string` | — | Sort direction (`asc`, `desc`). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `dataContracts` | `object[]` | Array of data contract objects. |
| `dataContracts[].id` | `string` | Contract identifier. |
| `dataContracts[].name` | `string` | Contract display name. |
| `dataContracts[].description` | `string` | Human-readable description. |
| `dataContracts[].producer` | `object` | Producer team details. |
| `dataContracts[].producer.team` | `string` | Producer team name. |
| `dataContracts[].producer.contactEmail` | `string` | Producer contact email. |
| `dataContracts[].consumer` | `object` | Consumer team details. |
| `dataContracts[].consumer.team` | `string` | Consumer team name. |
| `dataContracts[].consumer.contactEmail` | `string` | Consumer contact email. |
| `dataContracts[].table` | `string` | Fully-qualified table name. |
| `dataContracts[].status` | `string` | Contract status (`ACTIVE`, `DRAFT`, `VIOLATED`, `EXPIRED`, `ARCHIVED`). |
| `dataContracts[].version` | `string` | Semantic version. |
| `dataContracts[].schema` | `object[]` | Column definitions. |
| `dataContracts[].sla` | `object` | SLA thresholds. |
| `dataContracts[].qualityRules` | `object[]` | Quality rule definitions. |
| `dataContracts[].tags` | `object` | Key-value metadata tags. |
| `dataContracts[].lastVerifiedAt` | `string\|null` | ISO 8601 timestamp of last verification. |
| `dataContracts[].lastVerificationStatus` | `string\|null` | Last verification result (`PASSED`, `FAILED`). |
| `dataContracts[].violationCount` | `number` | Number of active violations. |
| `dataContracts[].expiresAt` | `string\|null` | ISO 8601 expiration timestamp. |
| `dataContracts[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `dataContracts[].updatedAt` | `string` | ISO 8601 last-update timestamp. |
| `dataContracts[].createdBy` | `string` | Creator identifier. |
| `totalCount` | `number` | Total number of contracts matching the filters. |

## Example

```bash title="cURL"
curl "/api/v1/data-contracts?status=ACTIVE&pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```
