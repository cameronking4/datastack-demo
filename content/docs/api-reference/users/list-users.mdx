---
title: List Users
description: Retrieve a paginated list of users.
---

```
GET /api/v1/users
```

Returns a paginated list of users with optional filters.

**Required scope:** `read:users`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number (min 1). |
| `perPage` | `number` | `20` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `data` | `object[]` | Array of user objects. |
| `data[].id` | `string` | User identifier. |
| `data[].fullName` | `string` | Full name. |
| `data[].email` | `string` | Email address. |
| `data[].avatarUrl` | `string` | Avatar image URL. |
| `data[].department` | `string` | Department name. |
| `data[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `data[].updatedAt` | `string` | ISO 8601 last update timestamp. |
| `data[].role` | `string` | User role (e.g., `admin`). |
| `data[].status` | `string` | Account status (`active`, `pending_verification`). |
| `data[].lastLoginAt` | `string` | ISO 8601 last login timestamp. |
| `pagination` | `object` | Pagination metadata. |
| `pagination.total` | `number` | Total number of users. |
| `pagination.page` | `number` | Current page. |
| `pagination.perPage` | `number` | Results per page. |
| `pagination.hasMore` | `boolean` | Whether more pages exist. |

## Example

```bash title="cURL"
curl "/api/v1/users?page=1&perPage=10" \
  -H "Authorization: Bearer <accessToken>"
```
