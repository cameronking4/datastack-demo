---
title: Get pipeline dependencies
description: Get upstream and downstream dependencies
---

```
GET /api/v1/pipelines/{pipelineId}/dependencies
```

Get upstream and downstream dependencies


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X GET "/api/v1/pipelines/example-pip/dependencies" \
  -H "Authorization: Bearer <accessToken>"
```
