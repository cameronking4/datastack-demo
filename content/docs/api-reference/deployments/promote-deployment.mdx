title: Promote deployment

---

`POST /api/v1/deployments/{deploymentId}/promote`

Promote a deployment to the next environment (dev -> staging -> production)

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `deploymentId` | `string` | Yes | The deploymentId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `targetEnvironment` | `string` | No | The targetEnvironment value. |

## Response `202`

Accepted

## Example

```bash title="cURL"
curl -X POST "/api/v1/deployments/{deploymentId}/promote" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "targetEnvironment": "example-targetEnvironment"
}'
```
