`GET /api/v1/pipelines/{pipelineId}/events`

List pipeline events

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/pipelines/{pipelineId}/events" \
  -H "Authorization: Bearer <accessToken>"
```
