---
title: List Alert Rules
description: List configured alert rules with trigger conditions and notification channels.
---

```
GET /api/v1/alerts
```

Returns a list of configured alert rules.

**Required scope:** `read:alerts`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `severity` | `string` | — | Filter by severity (`info`, `warning`, `critical`). |
| `enabled` | `boolean` | — | Filter by enabled state. |
| `resource` | `string` | — | Filter by resource identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `alerts` | `object[]` | Array of alert rule objects. |
| `alerts[].id` | `string` | Alert rule identifier. |
| `alerts[].name` | `string` | Alert rule name. |
| `alerts[].condition` | `object` | Trigger condition. |
| `alerts[].condition.metric` | `string` | Metric name. |
| `alerts[].condition.operator` | `string` | Comparison operator (`gt`, `gte`, `lt`, `lte`, `eq`, `neq`). |
| `alerts[].condition.threshold` | `number` | Threshold value. |
| `alerts[].condition.windowMinutes` | `integer` | Evaluation window in minutes. |
| `alerts[].severity` | `string` | Severity level. |
| `alerts[].channels` | `object[]` | Notification channels. |
| `alerts[].channels[].type` | `string` | Channel type (`email`, `slack`, `pagerduty`, `webhook`). |
| `alerts[].channels[].target` | `string` | Channel target. |
| `alerts[].enabled` | `boolean` | Whether the alert is enabled. |
| `totalCount` | `number` | Total number of alert rules. |

## Example

```bash title="cURL"
curl "/api/v1/alerts?severity=critical&enabled=true" \
  -H "Authorization: Bearer <accessToken>"
```
