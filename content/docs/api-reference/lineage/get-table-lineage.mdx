`GET /api/v1/lineage/table/{catalogName}/{schemaName}/{tableName}`

Get table lineage

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalogName` | `string` | Yes | The catalogName value. |
| `schemaName` | `string` | Yes | The schemaName value. |
| `tableName` | `string` | Yes | The tableName value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/lineage/table/{catalogName}/{schemaName}/{tableName}" \
  -H "Authorization: Bearer <accessToken>"
```
