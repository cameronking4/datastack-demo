`GET /api/v1/connections/{connectionId}`

Get connection details including health check status and credentials metadata

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `connectionId` | `string` | Yes | The connectionId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/connections/{connectionId}" \
  -H "Authorization: Bearer <accessToken>"
```
