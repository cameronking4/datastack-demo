---
title: Update Notification Channel
description: Update notification channel configuration, filters, or quiet hours.
---

```
PATCH /api/v1/notifications/{notificationId}
```

Update an existing notification channel.

**Required scope:** `manage:notifications`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `notificationId` | `string` | The notification channel identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | Display name. |
| `config` | `object` | No | Channel configuration. |
| `eventFilters` | `string[]` | No | Event type filters. |
| `severityFilter` | `string[]` | No | Severity level filters. |
| `enabled` | `boolean` | No | Whether the channel is active. |
| `quietHours` | `object` | No | Quiet hours configuration. |
| `rateLimitPerHour` | `number` | No | Maximum notifications per hour. |

## Response `200`

Returns the updated notification channel object.

## Example

```bash title="cURL"
curl -X PATCH /api/v1/notifications/notif-001 \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "enabled": false
  }'
```

```json title="Response"
{
  "id": "notif-001",
  "name": "Engineering Slack",
  "enabled": false,
  "updatedAt": "2024-06-10T12:00:00Z"
}
```
