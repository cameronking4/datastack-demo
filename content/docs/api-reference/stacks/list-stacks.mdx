`GET /api/v1/stacks`

List composable infrastructure stacks with module graph, lifecycle state, and compliance metadata

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `environment` | `string` | No | The environment value. |
| `lifecycleState` | `string` | No | The lifecycleState value. |
| `owner` | `string` | No | The owner value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/stacks" \
  -H "Authorization: Bearer <accessToken>"
```
