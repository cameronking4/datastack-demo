`GET /api/v1/run-metrics/{runId}`

Get run metrics

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `runId` | `string` | Yes | The runId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/run-metrics/{runId}" \
  -H "Authorization: Bearer <accessToken>"
```
