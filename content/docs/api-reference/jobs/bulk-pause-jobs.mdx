`POST /api/v1/jobs/bulk-pause`

Bulk pause jobs

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `jobIds` | `array<string>` | No | The jobIds value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/jobs/bulk-pause" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "jobIds": []
}'
```
