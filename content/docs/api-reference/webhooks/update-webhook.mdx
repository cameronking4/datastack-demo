---
title: Update webhook
description: Update webhook URL, events, retry policy, or signing configuration
---

`PATCH /api/v1/webhooks/{webhookId}`

Update webhook URL, events, retry policy, or signing configuration

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `webhookId` | `string` | Yes | The webhookId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `url` | `string` | No | The url value. |
| `events` | `array<string>` | No | The events value. |
| `signingSecret` | `string` | No | The signingSecret value. |
| `signingAlgorithm` | `string` | No | The signingAlgorithm value. |
| `active` | `boolean` | No | The active value. |
| `headers` | `object` | No | The headers value. |
| `retryPolicy` | `object` | No | The retryPolicy value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/webhooks/{webhookId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "url": "example-url",
  "events": [],
  "signingSecret": "example-signingSecret",
  "signingAlgorithm": "example-signingAlgorithm",
  "active": true,
  "headers": {},
  "retryPolicy": {}
}'
```
