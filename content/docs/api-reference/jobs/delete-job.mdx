`DELETE /api/v1/jobs/{jobId}`

Permanently delete a job and its run history

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/jobs/{jobId}" \
  -H "Authorization: Bearer <accessToken>"
```
