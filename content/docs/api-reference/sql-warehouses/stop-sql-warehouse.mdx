`POST /api/v1/sql/warehouses/{warehouseId}/stop`

Stop SQL warehouse

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `warehouseId` | `string` | Yes | The warehouseId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/sql/warehouses/{warehouseId}/stop" \
  -H "Authorization: Bearer <accessToken>"
```
