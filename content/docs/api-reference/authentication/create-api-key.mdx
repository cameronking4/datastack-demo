---
title: Create API Key
description: Create a new API key. The full key is returned only once.
---

```
POST /api/v1/auth/api-keys
```

Create a new API key with the specified name and scopes. The full key value is returned **only in this response** — store it securely.

**Required scope:** `manage:api_keys`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Display name for the key. |
| `scopes` | `string[]` | Yes | Permission scopes to grant. |
| `expiresInDays` | `number` | No | Days until expiration. Defaults to `90`. |
| `rateLimit` | `object` | No | Rate limiting configuration. |
| `rateLimit.requestsPerMinute` | `number` | No | Max requests per minute. |
| `rateLimit.requestsPerDay` | `number` | No | Max requests per day. |
| `rotationPolicy` | `object` | No | Automatic key rotation settings. |
| `rotationPolicy.autoRotate` | `boolean` | No | Enable automatic rotation. |
| `rotationPolicy.rotationIntervalDays` | `number` | No | Days between rotations. |
| `allowedIpRanges` | `string[]` | No | IP CIDR ranges allowed to use this key. |
| `metadata` | `object` | No | Arbitrary key-value metadata. |

## Response `201`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Key identifier. |
| `name` | `string` | Display name. |
| `key` | `string` | Full API key value (shown only once). |
| `prefix` | `string` | Key prefix for identification. |
| `scopes` | `string[]` | Granted scopes. |
| `expiresAt` | `string` | ISO 8601 expiration timestamp. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |

## Example

```bash title="cURL"
curl -X POST /api/v1/auth/api-keys \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "CI/CD Pipeline Key",
    "scopes": ["read:clusters", "manage:jobs"],
    "expiresInDays": 180
  }'
```

```json title="Response"
{
  "id": "key-new",
  "name": "CI/CD Pipeline Key",
  "key": "dsk_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  "prefix": "dsk_live_xxx",
  "scopes": ["read:clusters", "manage:jobs"],
  "expiresAt": "2025-12-01T00:00:00Z",
  "createdAt": "2024-06-01T10:00:00Z"
}
```
