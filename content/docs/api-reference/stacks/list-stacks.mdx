---
title: List Stacks
description: Retrieve a list of composable infrastructure stacks.
---

```
GET /api/v1/stacks
```

Returns a list of infrastructure stacks with module graph, lifecycle state, and compliance metadata.

**Required scope:** `read:stacks`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `environment` | `string` | — | Filter by environment (`development`, `staging`, `production`). |
| `lifecycleState` | `string` | — | Filter by lifecycle state (`DRAFT`, `ACTIVE`, `DEPLOYING`, `FAILED`, `DECOMMISSIONED`). |
| `owner` | `string` | — | Filter by owner identity. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `stacks` | `object[]` | Array of stack objects. |
| `stacks[].id` | `string` | Stack identifier. |
| `stacks[].name` | `string` | Display name. |
| `stacks[].description` | `string` | Stack description. |
| `stacks[].workspaceId` | `string` | Workspace the stack belongs to. |
| `stacks[].lifecycleState` | `string` | Lifecycle state (`DRAFT`, `ACTIVE`, `DEPLOYING`, `FAILED`, `DECOMMISSIONED`). |
| `stacks[].environment` | `string` | Target environment (`development`, `staging`, `production`). |
| `stacks[].owner` | `string` | Owner identity. |
| `stacks[].modules` | `object[]` | List of infrastructure modules. |
| `stacks[].modules[].name` | `string` | Module name. |
| `stacks[].modules[].provider` | `string` | Cloud provider. |
| `stacks[].modules[].resourceType` | `string` | Resource type. |
| `stacks[].modules[].resourceId` | `string` | Referenced resource identifier. |
| `stacks[].policies` | `object` | Stack governance policies. |
| `stacks[].compliance` | `object` | Compliance metadata. |
| `stacks[].revision` | `number` | Stack revision number. |
| `stacks[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `stacks[].updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `stacks[].createdBy` | `string` | Creator identity. |
| `totalCount` | `number` | Total number of stacks. |

## Example

```bash title="cURL"
curl "/api/v1/stacks?environment=production&lifecycleState=ACTIVE" \
  -H "Authorization: Bearer <accessToken>"
```
