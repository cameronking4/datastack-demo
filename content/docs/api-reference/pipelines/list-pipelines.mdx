`GET /api/v1/pipelines`

List Delta Live Tables pipelines with state and edition filters

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `state` | `string` | No | The state value. |
| `edition` | `string` | No | The edition value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/pipelines" \
  -H "Authorization: Bearer <accessToken>"
```
