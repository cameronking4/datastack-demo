`GET /api/v1/ml/feature-store/features`

List feature store features

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `featureGroup` | `string` | No | The featureGroup value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/ml/feature-store/features" \
  -H "Authorization: Bearer <accessToken>"
```
