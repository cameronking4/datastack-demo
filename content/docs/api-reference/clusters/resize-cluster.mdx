---
title: Resize Cluster
description: Adjust the worker count of a running cluster.
---

```
POST /api/v1/clusters/{clusterId}/resize
```

Resize a running cluster by setting a new worker count and optionally changing the node type.

**Required scope:** `manage:clusters`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `clusterId` | `string` | The cluster identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `workerCount` | `integer` | Yes | Desired number of worker nodes. |
| `nodeType` | `string` | No | Instance type for workers. |

## Response `200`

Returns a success response when the resize operation is accepted.

## Example

```bash title="cURL"
curl -X POST /api/v1/clusters/cluster-001/resize \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{"workerCount": 8}'
```

```json title="Response — 200 OK"
{
  "status": "ok"
}
```
