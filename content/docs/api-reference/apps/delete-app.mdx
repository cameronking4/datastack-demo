`DELETE /api/v1/apps/{appId}`

Delete app

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `appId` | `string` | Yes | The appId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/apps/{appId}" \
  -H "Authorization: Bearer <accessToken>"
```
