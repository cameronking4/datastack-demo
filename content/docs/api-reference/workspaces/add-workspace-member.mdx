---
title: Add Workspace Member
description: Add a member to a workspace with a specified role.
---

```
POST /api/v1/workspaces/{workspaceId}/members
```

Adds a user to a workspace with the specified role.

**Required scope:** `manage:workspaces`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `workspaceId` | `string` | Workspace identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `userId` | `string` | Yes | User identifier to add. |
| `role` | `string` | Yes | Role to assign (`admin`, `editor`, `viewer`). |
| `expiresAt` | `string` | No | ISO 8601 access expiration date-time. |

## Response `201`

Returns the created membership object.

## Example

```bash title="cURL"
curl -X POST "/api/v1/workspaces/ws-001/members" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "user-002",
    "role": "editor"
  }'
```

```json title="Response"
{
  "userId": "user-002",
  "workspaceId": "ws-001",
  "role": "editor",
  "joinedAt": "2024-06-10T10:00:00Z"
}
```
