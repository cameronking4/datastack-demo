`POST /api/v1/clusters/{clusterId}/upgrade`

Schedule runtime upgrade for a cluster

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `clusterId` | `string` | Yes | The clusterId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `targetRuntimeVersion` | `string` | No | The targetRuntimeVersion value. |
| `scheduleAt` | `string` | No | The scheduleAt value. |

## Response `202`

Accepted

## Example

```bash title="cURL"
curl -X POST "/api/v1/clusters/{clusterId}/upgrade" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "targetRuntimeVersion": "example-targetRuntimeVersion",
  "scheduleAt": "example-scheduleAt"
}'
```
