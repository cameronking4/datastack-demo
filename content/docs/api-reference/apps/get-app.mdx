`GET /api/v1/apps/{appId}`

Get app

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `appId` | `string` | Yes | The appId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/apps/{appId}" \
  -H "Authorization: Bearer <accessToken>"
```
