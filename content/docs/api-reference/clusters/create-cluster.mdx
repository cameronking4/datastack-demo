---
title: Create Cluster
description: Create a new compute cluster in your workspace.
---

```
POST /api/v1/clusters
```

Provision a new compute cluster.

**Required scope:** `manage:clusters`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Cluster display name. |
| `workspaceId` | `string` | Yes | Target workspace ID. |
| `runtimeVersion` | `string` | Yes | Spark runtime version (e.g., `3.5.x-scala2.12`). |
| `nodeType` | `string` | Yes | VM instance type (e.g., `i3.xlarge`). |
| `driverNodeType` | `string` | No | Driver node instance type. Defaults to same as `nodeType`. |
| `numWorkers` | `integer` | No | Number of workers. Defaults to `2`. |
| `enableAutoscaling` | `boolean` | No | Enable autoscaling. Defaults to `false`. |
| `minWorkers` | `integer` | No | Minimum workers when autoscaling. Defaults to `1`. |
| `maxWorkers` | `integer` | No | Maximum workers when autoscaling. Defaults to `8`. |
| `autoTerminationMinutes` | `integer` | No | Idle timeout in minutes. Defaults to `30`. |
| `spotInstances` | `boolean` | No | Use spot/preemptible instances. |
| `spotFallbackToOnDemand` | `boolean` | No | Fall back to on-demand if spot is unavailable. |
| `instancePoolId` | `string` | No | Instance pool to use for cluster nodes. |
| `clusterPolicyId` | `string` | No | Cluster policy to apply. |
| `runtimeEngine` | `string` | No | Runtime engine (`STANDARD`, `PHOTON`). |
| `initScripts` | `array` | No | Initialization scripts to run on startup. |
| `customSparkConf` | `object` | No | Custom Spark configuration key-value pairs. |
| `sshPublicKeys` | `array` | No | SSH public keys for node access. |
| `tags` | `object` | No | Key-value metadata tags. |

## Response `201`

Returns the created cluster object with state `PENDING`.

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Cluster identifier. |
| `name` | `string` | Display name. |
| `workspaceId` | `string` | Workspace identifier. |
| `clusterSize` | `string` | Capacity label. |
| `region` | `string` | Deployment region. |
| `state` | `string` | `PENDING` while provisioning. |
| `runtimeVersion` | `string` | Runtime version. |
| `nodeType` | `string` | Instance type. |
| `numWorkers` | `integer` | Number of workers. |
| `enableAutoscaling` | `boolean` | Autoscaling enabled. |
| `minWorkers` | `integer` | Minimum workers. |
| `maxWorkers` | `integer` | Maximum workers. |
| `autoTerminationMinutes` | `integer` | Idle timeout. |
| `tags` | `object` | Metadata tags. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `createdBy` | `string` | Creator identifier. |

## Example

```bash title="cURL"
curl -X POST /api/v1/clusters \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Analytics Cluster",
    "workspaceId": "ws-001",
    "runtimeVersion": "3.5.x-scala2.12",
    "nodeType": "i3.xlarge",
    "numWorkers": 4,
    "autoTerminationMinutes": 30
  }'
```
