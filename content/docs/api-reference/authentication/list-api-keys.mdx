---
title: List API Keys
description: List all API keys for the authenticated user.
---

```
GET /api/v1/auth/api-keys
```

Returns all API keys with usage statistics and expiration info.

**Required scope:** `manage:api_keys`

## Query Parameters

| Parameter | Type | Description |
|---|---|---|
| `status` | `string` | Filter by key status: `active`, `expired`, or `revoked`. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `apiKeys` | `object[]` | Array of API key objects. |
| `apiKeys[].id` | `string` | Key identifier. |
| `apiKeys[].name` | `string` | Display name. |
| `apiKeys[].prefix` | `string` | Key prefix for identification (e.g., `dsk_live_abc`). |
| `apiKeys[].scopes` | `string[]` | Granted permission scopes. |
| `apiKeys[].expiresAt` | `string` | ISO 8601 expiration timestamp. |
| `apiKeys[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `apiKeys[].lastUsedAt` | `string` | ISO 8601 timestamp of last use. |
| `apiKeys[].status` | `string` | Key status (`ACTIVE` or `REVOKED`). |
| `totalCount` | `number` | Total number of keys. |

## Example

```bash title="cURL"
curl /api/v1/auth/api-keys \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "apiKeys": [
    {
      "id": "key-001",
      "name": "CI/CD Pipeline Key",
      "prefix": "dsk_live_abc",
      "scopes": ["read:clusters", "manage:jobs"],
      "expiresAt": "2025-06-01T00:00:00Z",
      "createdAt": "2024-06-01T10:00:00Z",
      "lastUsedAt": "2024-06-10T08:00:00Z",
      "status": "ACTIVE"
    }
  ],
  "totalCount": 1
}
```
