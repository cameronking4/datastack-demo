`POST /api/v1/stacks`

Create a new composable infrastructure stack with module graph, policies, and drift detection

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `description` | `string` | No | The description value. |
| `workspaceId` | `string` | Yes | The workspaceId value. |
| `environment` | `string` | Yes | The environment value. |
| `owner` | `string` | Yes | The owner value. |
| `modules` | `array<object>` | Yes | The modules value. |
| `policies` | `object` | No | The policies value. |
| `compliance` | `object` | No | The compliance value. |
| `tags` | `object` | No | The tags value. |
| `gitRepository` | `string` | No | The gitRepository value. |
| `gitBranch` | `string` | No | The gitBranch value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/stacks" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "description": "example-description",
  "workspaceId": "example-workspaceId",
  "environment": "example-environment",
  "owner": "example-owner",
  "modules": [],
  "policies": {},
  "compliance": {},
  "tags": {},
  "gitRepository": "example-gitRepository",
  "gitBranch": "example-gitBranch"
}'
```
