title: Create webhook
description: Register a new webhook endpoint with event subscriptions and retry configuration
---

`POST /api/v1/webhooks`

Register a new webhook endpoint with event subscriptions and retry configuration

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `url` | `string` | Yes | The url value. |
| `events` | `array<string>` | Yes | The events value. |
| `signingSecret` | `string` | No | The signingSecret value. |
| `signingAlgorithm` | `string` | No | The signingAlgorithm value. |
| `active` | `boolean` | No | The active value. |
| `headers` | `object` | No | The headers value. |
| `retryPolicy` | `object` | No | The retryPolicy value. |
| `contentType` | `string` | No | The contentType value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/webhooks" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "url": "example-url",
  "events": [],
  "signingSecret": "example-signingSecret",
  "signingAlgorithm": "example-signingAlgorithm",
  "active": true,
  "headers": {},
  "retryPolicy": {},
  "contentType": "example-contentType"
}'
```
