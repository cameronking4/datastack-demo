---
title: Delete Secret Scope
description: Delete an entire secret scope and all its secrets.
---

```
DELETE /api/v1/secrets/scopes/{scopeName}
```

Delete a secret scope and all secrets it contains. This action is irreversible.

**Required scope:** `manage:secrets`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `scopeName` | `string` | The scope name. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `message` | `string` | Confirmation message. |

## Example

```bash title="cURL"
curl -X DELETE /api/v1/secrets/scopes/old-scope \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "message": "Secret scope 'old-scope' deleted."
}
```
