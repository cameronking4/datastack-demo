---
title: Get Schedule
description: Get schedule details including fire history and next execution time.
---

```
GET /api/v1/schedules/{scheduleId}
```

Returns full details for a single schedule, including next execution time.

**Required scope:** `read:schedules`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `scheduleId` | `string` | The schedule identifier. |

## Response `200`

Returns the schedule object.

## Example

```bash title="cURL"
curl /api/v1/schedules/sched-001 \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "id": "sched-001",
  "name": "Nightly ETL",
  "resourceType": "job",
  "resourceId": "job-001",
  "scheduleExpression": "0 2 * * *",
  "timezone": "America/New_York",
  "active": true,
  "skipIfRunning": true,
  "catchUpMissed": false,
  "createdAt": "2024-03-01T09:00:00Z"
}
```
