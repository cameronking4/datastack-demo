---
title: List Pipelines
description: Retrieve a paginated list of Delta Live Tables pipelines.
---

```
GET /api/v1/pipelines
```

Returns a paginated list of Delta Live Tables pipelines.

**Required scope:** `read:pipelines`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `pageSize` | `number` | `25` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `pipelines` | `object[]` | Array of pipeline objects. |
| `pipelines[].id` | `string` | Pipeline identifier. |
| `pipelines[].name` | `string` | Pipeline name. |
| `pipelines[].workspaceId` | `string` | Workspace identifier. |
| `pipelines[].state` | `string` | Current state (`RUNNING`, `IDLE`, `STOPPING`). |
| `pipelines[].target` | `string` | Target environment (`PRODUCTION`, `DEVELOPMENT`, `STAGING`). |
| `pipelines[].catalog` | `string` | Unity Catalog name. |
| `pipelines[].schema` | `string` | Database schema. |
| `pipelines[].continuous` | `boolean` | Whether the pipeline runs continuously. |
| `pipelines[].photon` | `boolean` | Whether Photon acceleration is enabled. |
| `pipelines[].edition` | `string` | Product edition (`CORE`, `PRO`, `ADVANCED`). |
| `pipelines[].clusters` | `object[]` | Associated cluster configurations. |
| `pipelines[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `pipelines[].createdBy` | `string` | Creator email. |
| `pipelines[].lastRunAt` | `string` | ISO 8601 timestamp of last run. |
| `totalCount` | `number` | Total number of pipelines. |

## Example

```bash title="cURL"
curl "/api/v1/pipelines?pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```
