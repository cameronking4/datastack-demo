`PATCH /api/v1/clusters/{clusterId}`

Update cluster configuration. Some changes require a restart.

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `clusterId` | `string` | Yes | The clusterId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `numWorkers` | `integer` | No | The numWorkers value. |
| `nodeType` | `string` | No | The nodeType value. |
| `driverNodeType` | `string` | No | The driverNodeType value. |
| `autoTerminationMinutes` | `integer` | No | The autoTerminationMinutes value. |
| `enableAutoscaling` | `boolean` | No | The enableAutoscaling value. |
| `minWorkers` | `integer` | No | The minWorkers value. |
| `maxWorkers` | `integer` | No | The maxWorkers value. |
| `spotInstances` | `boolean` | No | The spotInstances value. |
| `runtimeEngine` | `string` | No | The runtimeEngine value. |
| `customSparkConf` | `object` | No | The customSparkConf value. |
| `tags` | `object` | No | The tags value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/clusters/{clusterId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "numWorkers": 10,
  "nodeType": "example-nodeType",
  "driverNodeType": "example-driverNodeType",
  "autoTerminationMinutes": 10,
  "enableAutoscaling": true,
  "minWorkers": 10,
  "maxWorkers": 10,
  "spotInstances": true,
  "runtimeEngine": "example-runtimeEngine",
  "customSparkConf": {},
  "tags": {}
}'
```
