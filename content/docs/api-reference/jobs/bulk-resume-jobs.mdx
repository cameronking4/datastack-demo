---
title: Bulk Resume Jobs
description: Resume multiple paused jobs in a single request.
---

```
POST /api/v1/jobs/bulk-resume
```

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `jobIds` | `string[]` | Yes | Array of job identifiers to resume. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `resumed` | `string[]` | Job IDs that were resumed. |
| `failed` | `string[]` | Job IDs that failed to resume. |
| `message` | `string` | Summary message. |

## Example

```bash title="cURL"
curl -X POST "/api/v1/jobs/bulk-resume" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{"jobIds": ["job-001", "job-002"]}'
```
