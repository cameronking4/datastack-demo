`POST /api/v1/data-quality/monitors`

Create a monitor that runs quality rules on a schedule

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `table` | `string` | No | The table value. |
| `schedule` | `string` | No | The schedule value. |
| `ruleIds` | `array<string>` | No | The ruleIds value. |
| `alertOnFailure` | `boolean` | No | The alertOnFailure value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/data-quality/monitors" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "table": "example-table",
  "schedule": "example-schedule",
  "ruleIds": [],
  "alertOnFailure": true
}'
```
