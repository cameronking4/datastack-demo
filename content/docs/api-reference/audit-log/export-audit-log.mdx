```
POST /api/v1/audit-log/export
```

Request an audit log export (async job; download URL provided when ready)


## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `since` | `string` | No | The since value. Format: `date-time`. |
| `until` | `string` | No | The until value. Format: `date-time`. |
| `format` | `string` | No | The format value. One of: `json`, `csv`. |
| `resourceTypes` | `string[]` | No | The resourceTypes value. |

## Response `201`

Returns created.


## Example

```bash title="cURL"
curl -X POST "/api/v1/audit-log/export" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "since": "2024-01-01T00:00:00Z",
  "until": "2024-01-01T00:00:00Z",
  "format": "json",
  "resourceTypes": []
}'
```
