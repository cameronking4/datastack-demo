`PUT /api/v1/jobs/{jobId}/sla`

Update job SLA

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `maxDurationMinutes` | `integer` | No | The maxDurationMinutes value. |
| `deadlineCron` | `string` | No | The deadlineCron value. |
| `alertOnBreach` | `boolean` | No | The alertOnBreach value. |
| `notifyEmails` | `array<string>` | No | The notifyEmails value. |
| `priority` | `string` | No | The priority value. |
| `escalationPolicy` | `string` | No | The escalationPolicy value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PUT "/api/v1/jobs/{jobId}/sla" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "maxDurationMinutes": 10,
  "deadlineCron": "example-deadlineCron",
  "alertOnBreach": true,
  "notifyEmails": [],
  "priority": "example-priority",
  "escalationPolicy": "example-escalationPolicy"
}'
```
