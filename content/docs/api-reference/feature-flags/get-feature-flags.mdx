---
title: Get feature flags
description: Get feature flags and experiments for the current user or workspace
---

```
GET /api/v1/feature-flags
```

Get feature flags and experiments for the current user or workspace


## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | No | The workspaceId value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X GET "/api/v1/feature-flags" \
  -H "Authorization: Bearer <accessToken>"
```
