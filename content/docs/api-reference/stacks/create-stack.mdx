---
title: Create Stack
description: Create a new composable infrastructure stack.
---

```
POST /api/v1/stacks
```

Creates a new infrastructure stack with the specified modules, policies, and compliance metadata.

**Required scope:** `manage:stacks`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Stack display name. |
| `description` | `string` | No | Stack description. |
| `workspaceId` | `string` | Yes | Target workspace identifier. |
| `environment` | `string` | Yes | Target environment (`development`, `staging`, `production`). |
| `owner` | `string` | Yes | Owner identity (e.g., team or user). |
| `modules` | `object[]` | Yes | List of infrastructure modules. |
| `modules[].name` | `string` | Yes | Module name. |
| `modules[].provider` | `string` | Yes | Cloud provider (`aws`, `azure`, `gcp`, `datastack`, `custom`). |
| `modules[].resourceType` | `string` | Yes | Resource type. |
| `modules[].resourceId` | `string` | Yes | Referenced resource identifier. |
| `modules[].version` | `string` | No | Module version. |
| `modules[].inputs` | `object` | No | Module input parameters. |
| `modules[].outputs` | `string[]` | No | Module output names. |
| `modules[].dependsOn` | `string[]` | No | Names of modules this module depends on. |
| `modules[].healthCheck` | `object` | No | Health check configuration. |
| `modules[].healthCheck.enabled` | `boolean` | No | Enable health checking. |
| `modules[].healthCheck.endpoint` | `string` | No | Health check endpoint URL. |
| `modules[].healthCheck.intervalSeconds` | `integer` | No | Check interval in seconds. |
| `modules[].healthCheck.timeoutSeconds` | `integer` | No | Check timeout in seconds. |
| `policies` | `object` | No | Stack governance policies. |
| `policies.driftDetection` | `object` | No | Drift detection settings. |
| `policies.driftDetection.enabled` | `boolean` | No | Enable drift detection. |
| `policies.driftDetection.intervalMinutes` | `integer` | No | Detection interval in minutes. |
| `policies.driftDetection.autoRemediate` | `boolean` | No | Auto-fix detected drift. |
| `policies.rollback` | `object` | No | Rollback settings. |
| `policies.rollback.enabled` | `boolean` | No | Enable automatic rollback. |
| `policies.rollback.maxRevisions` | `integer` | No | Maximum revisions to retain. |
| `policies.approval` | `object` | No | Approval gate settings. |
| `policies.approval.required` | `boolean` | No | Require approval before deploy. |
| `policies.approval.approvers` | `string[]` | No | List of approver identities. |
| `compliance` | `object` | No | Compliance metadata. |
| `compliance.frameworks` | `string[]` | No | Compliance frameworks (e.g., `SOC2`, `HIPAA`). |
| `compliance.dataClassification` | `string` | No | Data classification (`public`, `internal`, `confidential`, `restricted`). |
| `tags` | `object` | No | Key-value tags. |
| `gitRepository` | `string` | No | Git repository URL for IaC source. |
| `gitBranch` | `string` | No | Git branch name. |

## Response `201`

Returns the created stack object.

## Example

```bash title="cURL"
curl -X POST "/api/v1/stacks" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "ML Feature Store",
    "workspaceId": "ws-001",
    "environment": "development",
    "owner": "data-eng",
    "modules": [
      {
        "name": "ingestion",
        "provider": "datastack",
        "resourceType": "pipeline",
        "resourceId": "pipe-002",
        "version": "1.0.0",
        "outputs": ["raw_table"]
      },
      {
        "name": "compute",
        "provider": "aws",
        "resourceType": "cluster",
        "resourceId": "cluster-002",
        "dependsOn": ["ingestion"]
      }
    ],
    "policies": {
      "driftDetection": { "enabled": true, "intervalMinutes": 30 },
      "rollback": { "enabled": true, "maxRevisions": 5 }
    },
    "compliance": {
      "frameworks": ["SOC2"],
      "dataClassification": "internal"
    }
  }'
```
