`GET /api/v1/stacks/{stackId}`

Get stack details including layers, version history, and lock state

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stackId` | `string` | Yes | The stackId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/stacks/{stackId}" \
  -H "Authorization: Bearer <accessToken>"
```
