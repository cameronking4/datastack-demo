description: Delete retention policy
---

```
DELETE /api/v1/retention/policies/{policyId}
```

Delete retention policy


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `policyId` | `string` | Yes | The policyId value. |

## Response `204`

Returns no content.


## Example

```bash title="cURL"
curl -X DELETE "/api/v1/retention/policies/example-pol" \
  -H "Authorization: Bearer <accessToken>"
```
