---
title: Get Environment
description: Get environment details including promotion rules, variables, and deployment count.
---

```
GET /api/v1/environments/{environmentId}
```

Returns full details for a single environment.

**Required scope:** `read:environments`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `environmentId` | `string` | The environment identifier. |

## Response `200`

Returns the environment object with promotion rules, variables, and deployment count.

## Example

```bash title="cURL"
curl /api/v1/environments/env-001 \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "id": "env-001",
  "name": "Production",
  "slug": "production",
  "workspaceId": "ws-001",
  "promotionRules": {
    "requiresApproval": true,
    "approvers": ["user-001"],
    "requiredChecks": ["tests", "lint"]
  },
  "variables": {
    "LOG_LEVEL": "warn"
  },
  "protected": true,
  "createdAt": "2024-03-01T09:00:00Z"
}
```
