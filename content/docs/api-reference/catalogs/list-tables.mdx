`GET /api/v1/catalogs/{catalogName}/schemas/{schemaName}/tables`

List tables in a schema

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalogName` | `string` | Yes | The catalogName value. |
| `schemaName` | `string` | Yes | The schemaName value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/catalogs/{catalogName}/schemas/{schemaName}/tables" \
  -H "Authorization: Bearer <accessToken>"
```
