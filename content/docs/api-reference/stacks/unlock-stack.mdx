---
title: Unlock stack
description: Release the exclusive lock on a stack
---

`POST /api/v1/stacks/{stackId}/unlock`

Release the exclusive lock on a stack

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stackId` | `string` | Yes | The stackId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `force` | `boolean` | No | The force value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/stacks/{stackId}/unlock" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "force": true
}'
```
