title: Create user
description: Create a new user or service account
---

`POST /api/v1/users`

Create a new user or service account

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `displayName` | `string` | Yes | The displayName value. |
| `email` | `string` | Yes | The email value. |
| `role` | `string` | Yes | The role value. |
| `department` | `string` | No | The department value. |
| `teamIds` | `array<string>` | No | The teamIds value. |
| `permissions` | `array<string>` | No | The permissions value. |
| `mfaEnabled` | `boolean` | No | The mfaEnabled value. |
| `externalId` | `string` | No | The externalId value. |
| `metadata` | `object` | No | The metadata value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/users" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "displayName": "example-displayName",
  "email": "example-email",
  "role": "example-role",
  "department": "example-department",
  "teamIds": [],
  "permissions": [],
  "mfaEnabled": true,
  "externalId": "example-externalId",
  "metadata": {}
}'
```
