---
title: Start Pipeline
description: Start a pipeline update.
---

```
POST /api/v1/pipelines/{pipelineId}/start
```

Start a pipeline, triggering a data update. The pipeline transitions to `RUNNING` state.

**Required scope:** `manage:pipelines`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `pipelineId` | `string` | The pipeline identifier. |

## Response `200`

Returns the pipeline object with state `RUNNING`.

## Example

```bash title="cURL"
curl -X POST /api/v1/pipelines/pipe-001/start \
  -H "Authorization: Bearer <accessToken>"
```
