---
title: List Webhook Deliveries
description: Retrieve delivery history for a webhook.
---

```
GET /api/v1/webhooks/{webhookId}/deliveries
```

Returns the delivery history for a webhook, including status codes and timing.

**Required scope:** `manage:webhooks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `webhookId` | `string` | The webhook identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `deliveries` | `object[]` | Array of delivery objects. |
| `deliveries[].id` | `string` | Delivery identifier. |
| `deliveries[].webhookId` | `string` | Webhook identifier. |
| `deliveries[].event` | `string` | Event type that triggered the delivery. |
| `deliveries[].requestUrl` | `string` | Target URL. |
| `deliveries[].responseStatus` | `number` | HTTP response status code. |
| `deliveries[].success` | `boolean` | Whether delivery succeeded. |
| `deliveries[].duration` | `number` | Round-trip time in milliseconds. |
| `deliveries[].timestamp` | `string` | ISO 8601 delivery timestamp. |
| `totalCount` | `number` | Total number of deliveries. |

## Example

```bash title="cURL"
curl /api/v1/webhooks/wh-001/deliveries \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "deliveries": [
    {
      "id": "del-001",
      "webhookId": "wh-001",
      "event": "job.completed",
      "requestUrl": "https://example.com/hooks/datastack",
      "responseStatus": 200,
      "success": true,
      "duration": 85,
      "timestamp": "2024-06-10T02:15:33Z"
    }
  ],
  "totalCount": 1
}
```
