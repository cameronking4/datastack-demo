```
POST /api/v1/dashboards
```

Create dashboard


## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `workspaceId` | `string` | No | The workspaceId value. |
| `widgets` | `any[]` | No | The widgets value. |

## Response `201`

Returns created.


## Example

```bash title="cURL"
curl -X POST "/api/v1/dashboards" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "workspaceId": "example-workspaceId",
  "widgets": []
}'
```
