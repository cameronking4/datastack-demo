`GET /api/v1/queries/slow`

List queries exceeding a duration threshold

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `thresholdMs` | `string` | No | The thresholdMs value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/queries/slow" \
  -H "Authorization: Bearer <accessToken>"
```
