---
title: Get Pipeline
description: Retrieve details of a specific pipeline.
---

```
GET /api/v1/pipelines/{pipelineId}
```

Returns full details for a single pipeline.

**Required scope:** `read:pipelines`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `pipelineId` | `string` | The pipeline identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Pipeline identifier. |
| `name` | `string` | Pipeline name. |
| `workspaceId` | `string` | Workspace identifier. |
| `state` | `string` | Current state. |
| `target` | `string` | Target environment. |
| `catalog` | `string` | Unity Catalog name. |
| `schema` | `string` | Database schema. |
| `continuous` | `boolean` | Continuous mode. |
| `photon` | `boolean` | Photon enabled. |
| `edition` | `string` | Product edition. |
| `clusters` | `object[]` | Cluster configurations. |
| `createdAt` | `string` | Creation timestamp. |
| `createdBy` | `string` | Creator email. |
| `lastRunAt` | `string` | Last run timestamp. |

## Example

```bash title="cURL"
curl /api/v1/pipelines/pipe-001 \
  -H "Authorization: Bearer <accessToken>"
```
