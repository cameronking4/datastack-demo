`POST /api/v1/pipelines/{pipelineId}/validate`

Validate pipeline configuration before starting (preview only)

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `fullRefresh` | `boolean` | No | The fullRefresh value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/pipelines/{pipelineId}/validate" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "fullRefresh": true
}'
```
