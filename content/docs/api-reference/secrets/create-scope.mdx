---
title: Create Secret Scope
description: Create a new secret scope.
---

```
POST /api/v1/secrets/scopes
```

Creates a new secret scope for organizing secrets.

**Required scope:** `manage:secrets`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Scope name. |
| `backendType` | `string` | No | Storage backend (default `DATASTACK`). |

## Response `201`

Returns the created scope object.

## Example

```bash title="cURL"
curl -X POST "/api/v1/secrets/scopes" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "name": "ml-prod" }'
```

```json title="Response"
{
  "name": "ml-prod",
  "backendType": "DATASTACK",
  "secretsCount": 0,
  "createdAt": "2024-06-10T10:00:00Z",
  "createdBy": "api"
}
```
