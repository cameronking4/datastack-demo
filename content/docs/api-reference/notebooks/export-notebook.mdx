`GET /api/v1/notebooks/export`

Export notebook at path in requested format (SOURCE, HTML, JUPYTER)

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/notebooks/export" \
  -H "Authorization: Bearer <accessToken>"
```
