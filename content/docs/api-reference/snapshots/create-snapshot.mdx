---
title: Create snapshot
description: Create a new table or volume snapshot
---

`POST /api/v1/snapshots`

Create a new table or volume snapshot

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |
| `resourceType` | `string` | Yes | The resourceType value. |
| `resourceId` | `string` | Yes | The resourceId value. |
| `name` | `string` | No | The name value. |
| `retentionDays` | `integer` | No | The retentionDays value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/snapshots" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "workspaceId": "example-workspaceId",
  "resourceType": "example-resourceType",
  "resourceId": "example-resourceId",
  "name": "example-name",
  "retentionDays": 10
}'
```
