---
title: Clone pipeline
description: Clone a pipeline with a new name (copy configuration)
---

```
POST /api/v1/pipelines/{pipelineId}/clone
```

Clone a pipeline with a new name (copy configuration)


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |

## Response `201`

Returns created.


## Example

```bash title="cURL"
curl -X POST "/api/v1/pipelines/example-pip/clone" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name"
}'
```
