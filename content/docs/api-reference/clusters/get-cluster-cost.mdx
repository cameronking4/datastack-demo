`GET /api/v1/clusters/{clusterId}/cost`

Get cost breakdown and recommendations for a cluster

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `clusterId` | `string` | Yes | The clusterId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/clusters/{clusterId}/cost" \
  -H "Authorization: Bearer <accessToken>"
```
