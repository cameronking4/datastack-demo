---
title: Create Secret Scope
description: Create a new secret scope.
---

```
POST /api/v1/secrets/scopes
```

Create a new secret scope for organizing secrets.

**Required scope:** `manage:secrets`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Scope name (must be unique). |
| `backendType` | `string` | No | Backend storage type. Defaults to `DATASTACK`. |

## Response `201`

Returns the created scope object.

| Field | Type | Description |
|---|---|---|
| `name` | `string` | Scope name. |
| `backendType` | `string` | Backend storage type. |
| `secretsCount` | `number` | Number of secrets (starts at `0`). |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `createdBy` | `string` | Creator identifier. |

## Example

```bash title="cURL"
curl -X POST /api/v1/secrets/scopes \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "name": "ml-staging" }'
```

```json title="Response"
{
  "name": "ml-staging",
  "backendType": "DATASTACK",
  "secretsCount": 0,
  "createdAt": "2024-06-10T10:00:00Z",
  "createdBy": "api"
}
```
