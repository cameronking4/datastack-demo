title: Update stack
description: Update stack configuration, layers, or git integration settings
---

`PATCH /api/v1/stacks/{stackId}`

Update stack configuration, layers, or git integration settings

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stackId` | `string` | Yes | The stackId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `description` | `string` | No | The description value. |
| `version` | `string` | No | The version value. |
| `layers` | `array<object>` | No | The layers value. |
| `connections` | `array<string>` | No | The connections value. |
| `tags` | `object` | No | The tags value. |
| `autoDeployOnPush` | `boolean` | No | The autoDeployOnPush value. |
| `gitRepository` | `string` | No | The gitRepository value. |
| `gitBranch` | `string` | No | The gitBranch value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/stacks/{stackId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "description": "example-description",
  "version": "example-version",
  "layers": [],
  "connections": [],
  "tags": {},
  "autoDeployOnPush": true,
  "gitRepository": "example-gitRepository",
  "gitBranch": "example-gitBranch"
}'
```
