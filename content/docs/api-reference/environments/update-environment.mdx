title: Update environment
description: Update environment settings, promotion rules, or access controls
---

`PATCH /api/v1/environments/{environmentId}`

Update environment settings, promotion rules, or access controls

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `environmentId` | `string` | Yes | The environmentId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `description` | `string` | No | The description value. |
| `color` | `string` | No | The color value. |
| `promotionRules` | `object` | No | The promotionRules value. |
| `accessControl` | `object` | No | The accessControl value. |
| `variables` | `object` | No | The variables value. |
| `protected` | `boolean` | No | The protected value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/environments/{environmentId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "description": "example-description",
  "color": "example-color",
  "promotionRules": {},
  "accessControl": {},
  "variables": {},
  "protected": true
}'
```
