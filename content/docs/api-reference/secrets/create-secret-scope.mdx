`POST /api/v1/secrets/scopes`

Create secret scope

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `backendType` | `string` | No | The backendType value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/secrets/scopes" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "backendType": "example-backendType"
}'
```
