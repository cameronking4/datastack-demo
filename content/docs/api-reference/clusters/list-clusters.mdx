`GET /api/v1/clusters`

List clusters in workspace with optional state and tag filters

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `state` | `string` | No | The state value. |
| `sortBy` | `string` | No | The sortBy value. |
| `sortOrder` | `string` | No | The sortOrder value. |
| `tagFilter` | `string` | No | The tagFilter value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/clusters" \
  -H "Authorization: Bearer <accessToken>"
```
