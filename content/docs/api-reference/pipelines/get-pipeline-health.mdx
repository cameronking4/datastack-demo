`GET /api/v1/pipelines/{pipelineId}/health`

Get pipeline health checks and status

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/pipelines/{pipelineId}/health" \
  -H "Authorization: Bearer <accessToken>"
```
