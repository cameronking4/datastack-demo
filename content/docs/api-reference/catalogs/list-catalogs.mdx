---
title: List Catalogs
description: Retrieve a list of Unity Catalog catalogs in your workspace.
---

```
GET /api/v1/catalogs
```

Returns a list of catalogs in the workspace.

**Required scope:** `read:catalogs`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `pageSize` | `number` | `25` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `catalogs` | `object[]` | Array of catalog objects. |
| `catalogs[].name` | `string` | Catalog name (unique identifier). |
| `catalogs[].owner` | `string` | Catalog owner. |
| `catalogs[].comment` | `string` | Catalog description. |
| `catalogs[].properties` | `object` | Key-value properties. |
| `catalogs[].isolation` | `string` | Isolation mode (`OPEN`, `ISOLATED`). |
| `catalogs[].schemasCount` | `number` | Number of schemas in the catalog. |
| `catalogs[].createdAt` | `string` | ISO 8601 creation timestamp. |
| `catalogs[].updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `catalogs[].createdBy` | `string` | Creator identifier. |
| `totalCount` | `number` | Total number of catalogs. |
| `pageSize` | `number` | Page size. |

## Example

```bash title="cURL"
curl "/api/v1/catalogs?pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "catalogs": [

      "name": "main",

      "comment": "Primary production catalog",

      "createdBy": "ada@datastack.dev"
    }
  ],
  "totalCount": 2,
  "pageSize": 10
}
```
