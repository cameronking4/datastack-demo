`POST /api/v1/catalogs/{catalogName}/schemas/{schemaName}/tables`

Create table

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalogName` | `string` | Yes | The catalogName value. |
| `schemaName` | `string` | Yes | The schemaName value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `tableType` | `string` | No | The tableType value. |
| `dataSourceFormat` | `string` | No | The dataSourceFormat value. |
| `columns` | `array<object>` | Yes | The columns value. |
| `storageLocation` | `string` | No | The storageLocation value. |
| `comment` | `string` | No | The comment value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/catalogs/{catalogName}/schemas/{schemaName}/tables" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "tableType": "example-tableType",
  "dataSourceFormat": "example-dataSourceFormat",
  "columns": [],
  "storageLocation": "example-storageLocation",
  "comment": "example-comment"
}'
```
