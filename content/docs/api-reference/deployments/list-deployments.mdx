---
title: List Deployments
description: List all stack deployments across environments.
---

```
GET /api/v1/deployments
```

Returns a paginated list of all stack deployments across environments.

**Required scope:** `read:deployments`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number (min 1). |
| `pageSize` | `number` | `25` | Results per page (1–100). |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `deployments` | `object[]` | Array of deployment objects. |
| `deployments[].id` | `string` | Deployment identifier. |
| `deployments[].stackId` | `string` | Stack identifier. |
| `deployments[].stackName` | `string` | Stack display name. |
| `deployments[].version` | `number` | Stack version deployed. |
| `deployments[].environment` | `string` | Target environment (`PRODUCTION`, `STAGING`, `DEVELOPMENT`). |
| `deployments[].status` | `string` | Deployment status (`SUCCEEDED`, `FAILED`, `IN_PROGRESS`). |
| `deployments[].layersDeployed` | `number` | Layers successfully deployed. |
| `deployments[].layersFailed` | `number` | Layers that failed. |
| `deployments[].startedAt` | `string` | ISO 8601 start timestamp. |
| `deployments[].completedAt` | `string\|null` | ISO 8601 completion timestamp. |
| `deployments[].startedBy` | `string` | Initiator identifier. |
| `deployments[].promotedFrom` | `string\|null` | ID of the deployment this was promoted from. |
| `totalCount` | `number` | Total number of deployments. |
| `page` | `number` | Current page. |
| `pageSize` | `number` | Page size. |

## Example

```bash title="cURL"
curl "/api/v1/deployments?page=1&pageSize=10" \
  -H "Authorization: Bearer <accessToken>"
```
