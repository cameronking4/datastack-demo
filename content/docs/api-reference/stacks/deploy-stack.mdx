`POST /api/v1/stacks/{stackId}/deploy`

Deploy all layers of a stack to the target environment

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stackId` | `string` | Yes | The stackId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `environment` | `string` | No | The environment value. |
| `dryRun` | `boolean` | No | The dryRun value. |
| `force` | `boolean` | No | The force value. |

## Response `202`

Accepted

## Example

```bash title="cURL"
curl -X POST "/api/v1/stacks/{stackId}/deploy" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "environment": "example-environment",
  "dryRun": true,
  "force": true
}'
```
