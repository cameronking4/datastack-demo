---
title: Update Connection
description: Update a connection configuration.
---

```
PATCH /api/v1/connections/{connectionId}
```

Update connection properties. The connection type cannot be changed.

**Required scope:** `manage:connections`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `connectionId` | `string` | The connection identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | Updated display name. |
| `host` | `string` | No | Updated host URI. |
| `secretScope` | `string` | No | Updated secret scope. |
| `properties` | `object` | No | Updated connection properties. |

## Response `200`

Returns the updated connection object.

## Example

```bash title="cURL"
curl -X PATCH /api/v1/connections/conn-001 \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "properties": { "region": "us-west-2" } }'
```
