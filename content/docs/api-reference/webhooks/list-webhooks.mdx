`GET /api/v1/webhooks`

List registered webhooks with delivery stats

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `active` | `boolean` | No | The active value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/webhooks" \
  -H "Authorization: Bearer <accessToken>"
```
