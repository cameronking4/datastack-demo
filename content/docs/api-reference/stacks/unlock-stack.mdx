---
title: Unlock Stack
description: Release the exclusive lock on a stack.
---

```
POST /api/v1/stacks/{stackId}/unlock
```

Release the exclusive lock on a stack, allowing modifications to resume.

**Required scope:** `manage:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | The stack identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `force` | `boolean` | No | Force unlock even if locked by another user. |

## Response `200`

Returns confirmation.

## Example

```bash title="cURL"
curl -X POST /api/v1/stacks/stack-001/unlock \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{}'
```

```json title="Response"
{
  "message": "Stack stack-001 unlocked."
}
```
