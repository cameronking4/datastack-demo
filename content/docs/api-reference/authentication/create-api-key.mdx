---

`POST /api/v1/auth/api-keys`

Create a new API key with scopes, expiration, and rate limiting

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `scopes` | `array<string>` | Yes | The scopes value. |
| `expiresInDays` | `integer` | No | The expiresInDays value. |
| `rateLimit` | `object` | No | The rateLimit value. |
| `rotationPolicy` | `object` | No | The rotationPolicy value. |
| `allowedIpRanges` | `array<string>` | No | The allowedIpRanges value. |
| `metadata` | `object` | No | The metadata value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/auth/api-keys" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "scopes": [],
  "expiresInDays": 10,
  "rateLimit": {},
  "rotationPolicy": {},
  "allowedIpRanges": [],
  "metadata": {}
}'
```
