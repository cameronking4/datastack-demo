---
title: Unlock Stack
description: Release the exclusive lock on a stack.
---

```
POST /api/v1/stacks/{stackId}/unlock
```

Release a previously acquired lock on a stack.

**Required scope:** `manage:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | The stack identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `force` | `boolean` | No | Force-unlock even if the lock is held by another user. Requires admin privileges. |

## Response `200`

Returns confirmation that the lock was released.

## Example

```bash title="cURL"
curl -X POST /api/v1/stacks/stack-001/unlock \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{}'
```

```json title="Response"
{
  "stackId": "stack-001",
  "locked": false,
  "unlockedAt": "2024-06-10T12:15:00Z"
}
```
