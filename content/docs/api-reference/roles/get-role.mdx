`GET /api/v1/roles/{roleId}`

Get role

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `roleId` | `string` | Yes | The roleId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/roles/{roleId}" \
  -H "Authorization: Bearer <accessToken>"
```
