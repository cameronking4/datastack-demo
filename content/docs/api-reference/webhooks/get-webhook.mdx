`GET /api/v1/webhooks/{webhookId}`

Get webhook configuration and delivery statistics

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `webhookId` | `string` | Yes | The webhookId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/webhooks/{webhookId}" \
  -H "Authorization: Bearer <accessToken>"
```
