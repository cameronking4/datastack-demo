title: DataStack
description: DataStack platform documentation — manage clusters, jobs, pipelines, notebooks, SQL warehouses, and more through a unified REST API.
---

DataStack is a unified data platform for provisioning and managing data infrastructure programmatically. Use the REST API to create compute clusters, orchestrate data pipelines, schedule jobs, and query data through managed SQL warehouses.
