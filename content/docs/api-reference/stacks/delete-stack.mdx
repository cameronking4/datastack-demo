`DELETE /api/v1/stacks/{stackId}`

Delete a stack. The stack must be unlocked and have no active deployments.

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `stackId` | `string` | Yes | The stackId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/stacks/{stackId}" \
  -H "Authorization: Bearer <accessToken>"
```
