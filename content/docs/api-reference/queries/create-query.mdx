---
title: Create Query
description: Save a new SQL query.
---

```
POST /api/v1/queries
```

Saves a new SQL query for later execution.

**Required scope:** `manage:queries`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Display name. |
| `description` | `string` | No | Query description. |
| `warehouseId` | `string` | Yes | Target warehouse identifier. |
| `sql` | `string` | Yes | SQL statement. |
| `tags` | `string[]` | No | Tags for categorization. |

## Response `201`

Returns the created query object.

## Example

```bash title="cURL"
curl -X POST "/api/v1/queries" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Revenue by Region",
    "description": "Monthly revenue breakdown",
    "warehouseId": "wh-001",
    "sql": "SELECT region, SUM(amount) FROM main.gold.orders GROUP BY 1",
    "tags": ["finance", "reporting"]
  }'
```

```json title="Response"
{
  "id": "query-new",
  "name": "Revenue by Region",
  "description": "Monthly revenue breakdown",
  "warehouseId": "wh-001",
  "sql": "SELECT region, SUM(amount) FROM main.gold.orders GROUP BY 1",
  "status": "SAVED",
  "tags": ["finance", "reporting"],
  "lastExecutedAt": null,
  "lastExecutionDurationMs": null,
  "createdAt": "2024-06-10T10:00:00Z",
  "updatedAt": "2024-06-10T10:00:00Z",
  "createdBy": "api"
}
```
