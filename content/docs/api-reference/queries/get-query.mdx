`GET /api/v1/queries/{queryId}`

Get saved query details including execution history

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `queryId` | `string` | Yes | The queryId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/queries/{queryId}" \
  -H "Authorization: Bearer <accessToken>"
```
