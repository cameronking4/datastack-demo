---
title: Get Notification Channel
description: Get notification channel details including delivery statistics.
---

```
GET /api/v1/notifications/{notificationId}
```

Returns full details for a single notification channel.

**Required scope:** `read:notifications`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `notificationId` | `string` | The notification channel identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Channel identifier. |
| `name` | `string` | Display name. |
| `type` | `string` | Channel type. |
| `config` | `object` | Channel configuration. |
| `eventFilters` | `string[]` | Event type filters. |
| `severityFilter` | `string[]` | Severity level filters. |
| `enabled` | `boolean` | Whether the channel is active. |
| `quietHours` | `object` | Quiet hours configuration. |
| `rateLimitPerHour` | `number` | Rate limit. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |

## Example

```bash title="cURL"
curl /api/v1/notifications/notif-001 \
  -H "Authorization: Bearer <accessToken>"
```
