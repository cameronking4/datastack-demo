---
title: Get Stack
description: Retrieve a stack by identifier with full layer details.
---

```
GET /api/v1/stacks/{stackId}
```

Returns a single stack with detailed layer status.

**Required scope:** `read:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | Stack identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Stack identifier. |
| `name` | `string` | Display name. |
| `description` | `string` | Stack description. |
| `workspaceId` | `string` | Workspace identifier. |
| `status` | `string` | Current status (`DRAFT`, `DEPLOYED`, `FAILED`). |
| `environment` | `string` | Target environment. |
| `layers` | `object[]` | Ordered resource layers with health status. |
| `layers[].name` | `string` | Layer name. |
| `layers[].resourceType` | `string` | Resource type. |
| `layers[].resourceId` | `string` | Referenced resource identifier. |
| `layers[].order` | `number` | Execution order. |
| `layers[].status` | `string` | Layer health status (`HEALTHY`, `DEGRADED`, `FAILED`). |
| `layers[].lastSyncAt` | `string` | ISO 8601 last sync timestamp. |
| `connections` | `string[]` | Connection identifiers. |
| `version` | `number` | Stack version. |
| `tags` | `object` | Key-value tags. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `createdBy` | `string` | Creator identity. |

```bash title="cURL"
curl "/api/v1/stacks/stack-001" \
  -H "Authorization: Bearer <accessToken>"
```
