---
title: Update Notification Channel
description: Update notification channel configuration, filters, or quiet hours.
---

```
PATCH /api/v1/notifications/{notificationId}
```

Update an existing notification channel's configuration.

**Required scope:** `manage:notifications`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `notificationId` | `string` | The notification channel identifier. |

## Request Body

| Field | Type | Description |
|---|---|---|
| `name` | `string` | Display name. |
| `config` | `object` | Type-specific configuration (recipients, webhookUrl, slackChannel). |
| `eventFilters` | `string[]` | Events to subscribe to. |
| `severityFilter` | `string[]` | Severity levels. |
| `enabled` | `boolean` | Whether the channel is active. |
| `quietHours` | `object` | Quiet hours configuration. |
| `rateLimitPerHour` | `number` | Maximum notifications per hour. |

## Response `200`

Returns the updated notification channel.

## Example

```bash title="cURL"
curl -X PATCH /api/v1/notifications/notif-001 \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "severityFilter": ["warning", "error", "critical"],
    "rateLimitPerHour": 30
  }'
```

```json title="Response"
{
  "message": "Notification channel notif-001 updated."
}
```
