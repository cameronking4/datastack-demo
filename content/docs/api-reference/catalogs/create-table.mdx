---
title: Create Table
description: Create a new table in a schema.
---

```
POST /api/v1/catalogs/{catalogName}/schemas/{schemaName}/tables
```

Creates a new table within the specified schema.

**Required scope:** `manage:catalogs`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `catalogName` | `string` | Catalog name. |
| `schemaName` | `string` | Schema name. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Table name. |
| `columns` | `object[]` | Yes | Column definitions. |
| `columns[].name` | `string` | Yes | Column name. |
| `columns[].type` | `string` | Yes | Column data type. |
| `columns[].nullable` | `boolean` | No | Whether nullable (default `true`). |
| `columns[].comment` | `string` | No | Column description. |
| `tableType` | `string` | No | Table type (default `MANAGED`). |
| `dataSourceFormat` | `string` | No | Storage format (default `DELTA`). |
| `storageLocation` | `string` | No | Custom storage path. |
| `comment` | `string` | No | Table description. |

## Response `201`

Returns the created table object.

## Example

```bash title="cURL"
curl -X POST "/api/v1/catalogs/main/schemas/bronze/tables" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "page_views",
    "columns": [
      { "name": "view_id", "type": "STRING", "nullable": false },
      { "name": "url", "type": "STRING", "nullable": false },
      { "name": "viewed_at", "type": "TIMESTAMP", "nullable": false }
    ]
  }'
```
