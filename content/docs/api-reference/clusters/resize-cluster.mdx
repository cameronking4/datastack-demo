---
title: Resize cluster
description: Resize cluster workers or node type
---

`POST /api/v1/clusters/{clusterId}/resize`

Resize cluster workers or node type

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `clusterId` | `string` | Yes | The clusterId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `numWorkers` | `integer` | No | The numWorkers value. |
| `minWorkers` | `integer` | No | The minWorkers value. |
| `maxWorkers` | `integer` | No | The maxWorkers value. |
| `nodeType` | `string` | No | The nodeType value. |

## Response `202`

Accepted

## Example

```bash title="cURL"
curl -X POST "/api/v1/clusters/{clusterId}/resize" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "numWorkers": 10,
  "minWorkers": 10,
  "maxWorkers": 10,
  "nodeType": "example-nodeType"
}'
```
