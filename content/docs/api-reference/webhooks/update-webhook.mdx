---
title: Update Webhook
description: Update an existing webhook's configuration.
---

```
PATCH /api/v1/webhooks/{webhookId}
```

Update one or more properties of an existing webhook.

**Required scope:** `manage:webhooks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `webhookId` | `string` | The webhook identifier. |

## Request Body

All fields are optional.

| Field | Type | Description |
|---|---|---|
| `url` | `string` | Delivery URL. |
| `events` | `string[]` | Event types to subscribe to. |
| `active` | `boolean` | Whether the webhook is active. |
| `secret` | `string` | Shared secret for verification. |

## Response `200`

Returns the updated webhook object.

## Example

```bash title="cURL"
curl -X PATCH /api/v1/webhooks/wh-001 \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{"events": ["job.completed", "job.failed", "pipeline.completed"]}'
```

```json title="Response"
{
  "id": "wh-001",
  "url": "https://example.com/hooks/datastack",
  "events": ["job.completed", "job.failed"],
  "active": true,
  "createdAt": "2024-03-01T09:00:00Z",
  "updatedAt": "2024-06-10T12:00:00Z",
  "failureCount": 0
}
```
