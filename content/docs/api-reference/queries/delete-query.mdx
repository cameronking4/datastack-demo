---
title: Delete Query
description: Delete a saved SQL query.
---

```
DELETE /api/v1/queries/{queryId}
```

Deletes a saved query.

**Required scope:** `manage:queries`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `queryId` | `string` | Query identifier. |

## Response `204`

No content.

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/queries/query-001" \
  -H "Authorization: Bearer <accessToken>"
```
