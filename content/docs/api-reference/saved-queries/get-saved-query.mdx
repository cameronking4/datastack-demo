```
GET /api/v1/saved-queries/{savedQueryId}
```

Get saved query


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `savedQueryId` | `string` | Yes | The savedQueryId value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X GET "/api/v1/saved-queries/example-sav" \
  -H "Authorization: Bearer <accessToken>"
```
