`GET /api/v1/tags`

List tags used across the workspace (tag taxonomy)

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `prefix` | `string` | No | The prefix value. |
| `page` | `integer` | No | The page value. |
| `pageSize` | `integer` | No | The pageSize value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/tags" \
  -H "Authorization: Bearer <accessToken>"
```
