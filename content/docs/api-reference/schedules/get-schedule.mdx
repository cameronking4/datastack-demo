`GET /api/v1/schedules/{scheduleId}`

Get schedule details including fire history and next execution time

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `scheduleId` | `string` | Yes | The scheduleId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/schedules/{scheduleId}" \
  -H "Authorization: Bearer <accessToken>"
```
