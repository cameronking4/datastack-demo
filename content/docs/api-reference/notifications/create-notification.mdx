---
title: Create Notification Channel
description: Register a new notification channel for alerts and event notifications.
---

```
POST /api/v1/notifications
```

Register a new notification channel for alerts and event notifications.

**Required scope:** `manage:notifications`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Display name. |
| `type` | `string` | Yes | Channel type: `email`, `slack`, `pagerduty`, `webhook`, or `teams`. |
| `config` | `object` | Yes | Type-specific configuration. |
| `config.recipients` | `string[]` | No | Email recipients (for `email` type). |
| `config.webhookUrl` | `string` | No | Webhook URL (for `webhook` type). |
| `config.slackChannel` | `string` | No | Slack channel (for `slack` type). |
| `config.pagerdutyServiceKey` | `string` | No | PagerDuty service key (for `pagerduty` type). |
| `eventFilters` | `string[]` | No | Events to subscribe to. |
| `severityFilter` | `string[]` | No | Severity levels: `info`, `warning`, `error`, `critical`. |
| `enabled` | `boolean` | No | Whether the channel is active. Defaults to `true`. |
| `quietHours` | `object` | No | Suppress notifications during quiet hours. |
| `quietHours.enabled` | `boolean` | No | Enable quiet hours. |
| `quietHours.startTime` | `string` | No | Start time (HH:MM). |
| `quietHours.endTime` | `string` | No | End time (HH:MM). |
| `quietHours.timezone` | `string` | No | IANA timezone. |
| `rateLimitPerHour` | `number` | No | Maximum notifications per hour. |

## Response `201`

Returns the created notification channel.

## Example

```bash title="cURL"
curl -X POST /api/v1/notifications \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Ops Slack",
    "type": "slack",
    "config": {
      "slackChannel": "#ops-alerts"
    },
    "severityFilter": ["error", "critical"]
  }'
```

```json title="Response"
{
  "id": "notif-001",
  "name": "Ops Slack",
  "type": "slack",
  "enabled": true,
  "createdAt": "2024-06-01T10:00:00Z"
}
```
