`DELETE /api/v1/retention/policies/{policyId}`

Delete retention policy

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `policyId` | `string` | Yes | The policyId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/retention/policies/{policyId}" \
  -H "Authorization: Bearer <accessToken>"
```
