```
POST /api/v1/streams/{streamId}/checkpoint
```

Create or manage stream checkpoint


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `streamId` | `string` | Yes | The streamId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `action` | `string` | No | The action value. |

## Response `201`

Returns created.


## Example

```bash title="cURL"
curl -X POST "/api/v1/streams/example-str/checkpoint" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "action": "example-action"
}'
```
