title: Update connection
description: Update connection configuration, credentials, or health check settings
---

`PATCH /api/v1/connections/{connectionId}`

Update connection configuration, credentials, or health check settings

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `connectionId` | `string` | Yes | The connectionId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `host` | `string` | No | The host value. |
| `port` | `integer` | No | The port value. |
| `authMethod` | `string` | No | The authMethod value. |
| `credentialRef` | `string` | No | The credentialRef value. |
| `healthCheck` | `object` | No | The healthCheck value. |
| `properties` | `object` | No | The properties value. |
| `tags` | `object` | No | The tags value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/connections/{connectionId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "host": "example-host",
  "port": 10,
  "authMethod": "example-authMethod",
  "credentialRef": "example-credentialRef",
  "healthCheck": {},
  "properties": {},
  "tags": {}
}'
```
