`POST /api/v1/ml/models/{modelId}/deploy`

Deploy model

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `modelId` | `string` | Yes | The modelId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `environment` | `string` | No | The environment value. |
| `instanceType` | `string` | No | The instanceType value. |
| `replicas` | `integer` | No | The replicas value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/ml/models/{modelId}/deploy" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "environment": "example-environment",
  "instanceType": "example-instanceType",
  "replicas": 10
}'
```
