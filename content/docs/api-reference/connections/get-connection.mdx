---
title: Get Connection
description: Retrieve details of a specific connection.
---

```
GET /api/v1/connections/{connectionId}
```

Returns full details for a single connection.

**Required scope:** `read:connections`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `connectionId` | `string` | The connection identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Connection identifier. |
| `name` | `string` | Display name. |
| `type` | `string` | Connection type. |
| `host` | `string` | Host URI. |
| `status` | `string` | Current status. |
| `workspaceId` | `string` | Workspace identifier. |
| `secretScope` | `string` | Secret scope for credentials. |
| `properties` | `object` | Connection-specific properties. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `createdBy` | `string` | Creator identifier. |
| `lastTestedAt` | `string` | Last connectivity test timestamp. |
| `lastTestStatus` | `string` | Last test result. |

## Example

```bash title="cURL"
curl /api/v1/connections/conn-001 \
  -H "Authorization: Bearer <accessToken>"
```
