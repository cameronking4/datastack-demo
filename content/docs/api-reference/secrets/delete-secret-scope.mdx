`DELETE /api/v1/secrets/scopes/{scopeName}`

Delete secret scope

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `scopeName` | `string` | Yes | The scopeName value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/secrets/scopes/{scopeName}" \
  -H "Authorization: Bearer <accessToken>"
```
