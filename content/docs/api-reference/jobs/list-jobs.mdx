`GET /api/v1/jobs`

List jobs with optional filters by state, owner, and tags

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `state` | `string` | No | The state value. |
| `ownerId` | `string` | No | The ownerId value. |
| `tagFilter` | `string` | No | The tagFilter value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/jobs" \
  -H "Authorization: Bearer <accessToken>"
```
