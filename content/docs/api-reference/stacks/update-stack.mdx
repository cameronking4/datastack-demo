---
title: Update Stack
description: Update a stack configuration.
---

```
PATCH /api/v1/stacks/{stackId}
```

Update stack properties, layers, or connections. Increments the stack version.

**Required scope:** `manage:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | The stack identifier. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | Updated display name. |
| `description` | `string` | No | Updated description. |
| `layers` | `object[]` | No | Updated list of resource layers. |
| `connections` | `string[]` | No | Updated connection IDs. |
| `tags` | `object` | No | Updated metadata tags. |

## Response `200`

Returns the updated stack object with an incremented version number.

## Example

```bash title="cURL"
curl -X PATCH /api/v1/stacks/stack-001 \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "layers": [
      { "name": "ingestion", "resourceType": "pipeline", "resourceId": "pipe-001", "order": 1 },
      { "name": "compute", "resourceType": "cluster", "resourceId": "cluster-002", "order": 2 }
    ]
  }'
```
