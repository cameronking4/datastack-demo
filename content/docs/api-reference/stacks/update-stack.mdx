---
title: Update Stack
description: Update a stack configuration.
---

```
PATCH /api/v1/stacks/{stackId}
```

Updates one or more fields of an existing stack.

**Required scope:** `manage:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | Stack identifier. |

## Request Body

All fields are optional. Only provided fields are updated.

| Field | Type | Description |
|---|---|---|
| `name` | `string` | New display name. |
| `description` | `string` | New description. |
| `layers` | `object[]` | Replacement layer list. |
| `connections` | `string[]` | Replacement connection list. |
| `tags` | `object` | Replacement key-value tags. |

## Response `200`

Returns the updated stack object.

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/stacks/stack-001" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Analytics Platform v2" }'
```

{

  "workspaceId": "ws-001",
