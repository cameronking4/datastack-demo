`DELETE /api/v1/saved-queries/{savedQueryId}`

Delete saved query

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `savedQueryId` | `string` | Yes | The savedQueryId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/saved-queries/{savedQueryId}" \
  -H "Authorization: Bearer <accessToken>"
```
