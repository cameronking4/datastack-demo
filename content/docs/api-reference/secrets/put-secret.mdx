`PUT /api/v1/secrets`

Create or update a secret in a scope

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `scope` | `string` | Yes | The scope value. |
| `key` | `string` | Yes | The key value. |
| `value` | `string` | Yes | The value value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PUT "/api/v1/secrets" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "scope": "example-scope",
  "key": "example-key",
  "value": "example-value"
}'
```
