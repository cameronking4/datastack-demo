`PATCH /api/v1/jobs/{jobId}`

Update job configuration, schedule, or retry policy

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | No | The name value. |
| `description` | `string` | No | The description value. |
| `schedule` | `string` | No | The schedule value. |
| `timezone` | `string` | No | The timezone value. |
| `triggerType` | `string` | No | The triggerType value. |
| `maxConcurrentRuns` | `integer` | No | The maxConcurrentRuns value. |
| `retryPolicy` | `object` | No | The retryPolicy value. |
| `alertDestinations` | `array<object>` | No | The alertDestinations value. |
| `settings` | `object` | No | The settings value. |
| `state` | `string` | No | The state value. |
| `tags` | `object` | No | The tags value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/jobs/{jobId}" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "description": "example-description",
  "schedule": "example-schedule",
  "timezone": "example-timezone",
  "triggerType": "example-triggerType",
  "maxConcurrentRuns": 10,
  "retryPolicy": {},
  "alertDestinations": [],
  "settings": {},
  "state": "example-state",
  "tags": {}
}'
```
