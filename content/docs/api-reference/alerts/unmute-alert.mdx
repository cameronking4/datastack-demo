`DELETE /api/v1/alerts/{alertId}/mute`

Unmute alert

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `alertId` | `string` | Yes | The alertId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/alerts/{alertId}/mute" \
  -H "Authorization: Bearer <accessToken>"
```
