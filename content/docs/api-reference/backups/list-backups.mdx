`GET /api/v1/backups`

List workspace backups with optional filters by workspace and status

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | No | The workspaceId value. |
| `status` | `string` | No | The status value. |
| `limit` | `integer` | No | The limit value. |
| `offset` | `integer` | No | The offset value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/backups" \
  -H "Authorization: Bearer <accessToken>"
```
