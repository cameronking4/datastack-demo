```
GET /api/v1/lineage/column/{catalogName}/{schemaName}/{tableName}/{columnName}
```

Get column lineage


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `catalogName` | `string` | Yes | The catalogName value. |
| `schemaName` | `string` | Yes | The schemaName value. |
| `tableName` | `string` | Yes | The tableName value. |
| `columnName` | `string` | Yes | The columnName value. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X GET "/api/v1/lineage/column/example-cat/example-sch/example-tab/example-col" \
  -H "Authorization: Bearer <accessToken>"
```
