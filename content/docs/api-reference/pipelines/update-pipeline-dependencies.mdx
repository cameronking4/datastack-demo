`PUT /api/v1/pipelines/{pipelineId}/dependencies`

Update pipeline dependencies

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `pipelineId` | `string` | Yes | The pipelineId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PUT "/api/v1/pipelines/{pipelineId}/dependencies" \
  -H "Authorization: Bearer <accessToken>"
```
