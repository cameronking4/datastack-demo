---
title: Revoke Token
description: Revoke an access or refresh token.
---

```
POST /api/v1/auth/token/revoke
```

Revoke an active access token or refresh token. Once revoked, the token can no longer be used for authentication.

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `token` | `string` | Yes | The access or refresh token to revoke. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `revoked` | `boolean` | `true` when the token has been revoked. |

## Example

```bash title="cURL"
curl -X POST /api/v1/auth/token/revoke \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <accessToken>" \
  -d '{"token": "eyJhbGciOiJSUzI1NiIs..."}'
```

```json title="Response"
{
  "revoked": true
}
```
