`POST /api/v1/pipelines`

Create a new Delta Live Tables pipeline with notification and library configuration

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `workspaceId` | `string` | Yes | The workspaceId value. |
| `target` | `string` | Yes | The target value. |
| `catalog` | `string` | No | The catalog value. |
| `schema` | `string` | No | The schema value. |
| `continuous` | `boolean` | No | The continuous value. |
| `photon` | `boolean` | No | The photon value. |
| `edition` | `string` | No | The edition value. |
| `developmentMode` | `boolean` | No | The developmentMode value. |
| `channel` | `string` | No | The channel value. |
| `notifications` | `array<object>` | No | The notifications value. |
| `libraries` | `array<object>` | No | The libraries value. |
| `clusterConfig` | `object` | No | The clusterConfig value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/pipelines" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "workspaceId": "example-workspaceId",
  "target": "example-target",
  "catalog": "example-catalog",
  "schema": "example-schema",
  "continuous": true,
  "photon": true,
  "edition": "example-edition",
  "developmentMode": true,
  "channel": "example-channel",
  "notifications": [],
  "libraries": [],
  "clusterConfig": {}
}'
```
