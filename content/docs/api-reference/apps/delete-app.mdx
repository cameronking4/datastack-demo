---

```
DELETE /api/v1/apps/{appId}
```

Delete app


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `appId` | `string` | Yes | The appId value. |

## Response `204`

Returns no content.


## Example

```bash title="cURL"
curl -X DELETE "/api/v1/apps/example-app" \
  -H "Authorization: Bearer <accessToken>"
```
