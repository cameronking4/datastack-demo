---
title: Get Stack
description: Retrieve details of a specific stack including layer status.
---

```
GET /api/v1/stacks/{stackId}
```

Returns full details for a single stack including per-layer health status.

**Required scope:** `read:stacks`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `stackId` | `string` | The stack identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Stack identifier. |
| `name` | `string` | Display name. |
| `description` | `string` | Stack description. |
| `workspaceId` | `string` | Workspace identifier. |
| `status` | `string` | Current status. |
| `environment` | `string` | Target environment. |
| `layers` | `object[]` | Resource layers with health status. |
| `layers[].name` | `string` | Layer name. |
| `layers[].resourceType` | `string` | Resource type. |
| `layers[].resourceId` | `string` | Referenced resource ID. |
| `layers[].order` | `number` | Execution order. |
| `layers[].status` | `string` | Layer health (`HEALTHY`, `DEGRADED`, `FAILED`). |
| `layers[].lastSyncAt` | `string` | Last sync timestamp. |
| `connections` | `string[]` | Connection IDs. |
| `version` | `number` | Stack version number. |
| `tags` | `object` | Metadata tags. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `createdBy` | `string` | Creator identifier. |

## Example

```bash title="cURL"
curl /api/v1/stacks/stack-001 \
  -H "Authorization: Bearer <accessToken>"
```
