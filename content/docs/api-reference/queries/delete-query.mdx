---

`DELETE /api/v1/queries/{queryId}`

Permanently delete a saved query

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `queryId` | `string` | Yes | The queryId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/queries/{queryId}" \
  -H "Authorization: Bearer <accessToken>"
```
