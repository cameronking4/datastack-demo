`GET /api/v1/auth/api-keys`

List API keys with usage statistics and expiration info

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `status` | `string` | No | The status value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/auth/api-keys" \
  -H "Authorization: Bearer <accessToken>"
```
