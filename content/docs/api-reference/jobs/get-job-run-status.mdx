`GET /api/v1/jobs/{jobId}/runs/{runId}`

Get job run status

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |
| `runId` | `string` | Yes | The runId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/jobs/{jobId}/runs/{runId}" \
  -H "Authorization: Bearer <accessToken>"
```
