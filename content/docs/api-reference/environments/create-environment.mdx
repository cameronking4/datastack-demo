`POST /api/v1/environments`

Create a new deployment environment with approval and promotion configuration

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name value. |
| `slug` | `string` | Yes | The slug value. |
| `workspaceId` | `string` | Yes | The workspaceId value. |
| `description` | `string` | No | The description value. |
| `color` | `string` | No | The color value. |
| `promotionRules` | `object` | No | The promotionRules value. |
| `accessControl` | `object` | No | The accessControl value. |
| `variables` | `object` | No | The variables value. |
| `protected` | `boolean` | No | The protected value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/environments" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "name": "example-name",
  "slug": "example-slug",
  "workspaceId": "example-workspaceId",
  "description": "example-description",
  "color": "example-color",
  "promotionRules": {},
  "accessControl": {},
  "variables": {},
  "protected": true
}'
```
