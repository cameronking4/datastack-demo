`GET /api/v1/workspaces`

List workspaces with member counts and resource quotas

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `region` | `string` | No | The region value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/workspaces" \
  -H "Authorization: Bearer <accessToken>"
```
