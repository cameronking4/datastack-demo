```
DELETE /api/v1/dashboards/{dashboardId}
```

Delete dashboard


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `dashboardId` | `string` | Yes | The dashboardId value. |

## Response `204`

Returns no content.


## Example

```bash title="cURL"
curl -X DELETE "/api/v1/dashboards/example-das" \
  -H "Authorization: Bearer <accessToken>"
```
