---
title: Remove favorite
description: Remove a resource from favorites
---

```
DELETE /api/v1/favorites/{resourceType}/{resourceId}
```

Remove a resource from favorites


## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `resourceType` | `string` | Yes | The resourceType value. |
| `resourceId` | `string` | Yes | The resourceId value. |

## Response `204`

Returns no content.


## Example

```bash title="cURL"
curl -X DELETE "/api/v1/favorites/example-res/example-res" \
  -H "Authorization: Bearer <accessToken>"
```
