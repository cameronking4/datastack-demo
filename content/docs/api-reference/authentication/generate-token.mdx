---
title: Generate Token
description: Exchange credentials for an access token using the OAuth2 client_credentials grant.
---

```
POST /api/v1/auth/token
```

Exchange client credentials for a short-lived access token.

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `grantType` | `string` | No | Grant type. Use `client_credentials`. |
| `scopes` | `string[]` | No | Requested permission scopes. Defaults to `["read:users", "read:workspaces"]`. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `accessToken` | `string` | JWT access token. |
| `refreshToken` | `string` | Token used to obtain a new access token. |
| `tokenType` | `string` | Always `Bearer`. |
| `expiresIn` | `number` | Token lifetime in seconds (3600). |
| `scopes` | `string[]` | Granted scopes. |

## Example

```bash title="cURL"
curl -X POST /api/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "grantType": "client_credentials",
    "scopes": ["read:clusters", "manage:jobs"]
  }'
```

```json title="Response"
{
  "accessToken": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9.demo-token",
  "refreshToken": "drt_refresh_demo_token",
  "tokenType": "Bearer",
  "expiresIn": 3600,
  "scopes": ["read:clusters", "manage:jobs"]
}
```
