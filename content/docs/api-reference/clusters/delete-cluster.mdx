---
title: Delete Cluster
description: Terminate and delete a cluster.
---

```
DELETE /api/v1/clusters/{clusterId}
```

Initiate termination of a cluster. The cluster will be shut down and removed.

**Required scope:** `manage:clusters`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `clusterId` | `string` | The cluster identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `message` | `string` | Confirmation message. |

## Example

```bash title="cURL"
curl -X DELETE /api/v1/clusters/cluster-001 \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "message": "Cluster cluster-001 termination initiated."
}
```
