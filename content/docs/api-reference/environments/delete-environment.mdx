---
title: Delete Environment
description: Delete an environment. Protected environments cannot be deleted.
---

```
DELETE /api/v1/environments/{environmentId}
```

Permanently deletes an environment. Protected environments must be unprotected first.

**Required scope:** `manage:environments`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `environmentId` | `string` | Environment identifier. |

## Response `204`

No content.

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/environments/env-002" \
  -H "Authorization: Bearer <accessToken>"
```
