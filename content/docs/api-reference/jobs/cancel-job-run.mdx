`POST /api/v1/jobs/{jobId}/runs/{runId}/cancel`

Cancel job run

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |
| `runId` | `string` | Yes | The runId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `reason` | `string` | No | The reason value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/jobs/{jobId}/runs/{runId}/cancel" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "reason": "example-reason"
}'
```
