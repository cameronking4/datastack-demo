`POST /api/v1/workspaces/{workspaceId}/members`

Invite a user to the workspace with a specified role

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `email` | `string` | Yes | The email value. |
| `role` | `string` | Yes | The role value. |
| `expiresAt` | `string` | No | The expiresAt value. |

## Response `201`

Created

## Example

```bash title="cURL"
curl -X POST "/api/v1/workspaces/{workspaceId}/members" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "email": "example-email",
  "role": "example-role",
  "expiresAt": "example-expiresAt"
}'
```
