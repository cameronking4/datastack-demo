`PATCH /api/v1/workspaces/{workspaceId}/settings`

Update workspace settings

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `workspaceId` | `string` | Yes | The workspaceId value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X PATCH "/api/v1/workspaces/{workspaceId}/settings" \
  -H "Authorization: Bearer <accessToken>"
```
