```
GET /api/v1/health
```

Liveness/readiness check for load balancers and orchestrators. Query param probe=readiness includes dependency checks.


## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `probe` | `string` | No | The probe value. One of: `readiness`. |

## Response `200`

Returns success.


## Example

```bash title="cURL"
curl -X GET "/api/v1/health" \
  -H "Authorization: Bearer <accessToken>"
```
