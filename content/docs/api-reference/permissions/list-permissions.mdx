`GET /api/v1/permissions`

List available permissions with optional scope filter

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `scope` | `string` | No | The scope value. |
| `page` | `integer` | No | The page value. |
| `pageSize` | `integer` | No | The pageSize value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X GET "/api/v1/permissions" \
  -H "Authorization: Bearer <accessToken>"
```
