`DELETE /api/v1/roles/{roleId}`

Delete role

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `roleId` | `string` | Yes | The roleId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/roles/{roleId}" \
  -H "Authorization: Bearer <accessToken>"
```
