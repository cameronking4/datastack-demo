`DELETE /api/v1/schedules/{scheduleId}`

Delete a schedule. The associated resource will no longer be triggered automatically.

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `scheduleId` | `string` | Yes | The scheduleId value. |

## Response `204`

No content

## Example

```bash title="cURL"
curl -X DELETE "/api/v1/schedules/{scheduleId}" \
  -H "Authorization: Bearer <accessToken>"
```
