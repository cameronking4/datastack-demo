---
title: List Audit Log Entries
description: Query the audit log for user and system actions.
---

```
GET /api/v1/audit-log
```

Query the audit log for user and system actions with filtering by actor, action, resource, and time range.

**Required scope:** `read:audit_log`

## Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `actor` | `string` | — | Filter by actor identity. |
| `action` | `string` | — | Filter by action (`create`, `update`, `delete`, `deploy`, `execute`, `login`, `logout`, `invite`, `revoke`). |
| `resourceType` | `string` | — | Filter by resource type (`pipeline`, `cluster`, `stack`, `connection`, `secret`, `webhook`, `user`, `api_key`). |
| `resourceId` | `string` | — | Filter by resource identifier. |
| `since` | `string` | — | Start of time range (ISO 8601 date-time). |
| `until` | `string` | — | End of time range (ISO 8601 date-time). |
| `limit` | `integer` | `25` | Results per page. |
| `offset` | `integer` | `0` | Number of results to skip. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `entries` | `object[]` | Array of audit log entries. |
| `entries[].id` | `string` | Entry identifier. |
| `entries[].actor` | `string` | Identity that performed the action. |
| `entries[].action` | `string` | Action performed. |
| `entries[].resourceType` | `string` | Type of resource affected. |
| `entries[].resourceId` | `string` | Identifier of the affected resource. |
| `entries[].timestamp` | `string` | ISO 8601 timestamp. |
| `entries[].metadata` | `object` | Additional context for the action. |
| `totalCount` | `number` | Total matching entries. |

## Example

```bash title="cURL"
curl "/api/v1/audit-log?action=deploy&resourceType=stack&since=2024-06-01T00:00:00Z&limit=10" \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "entries": [
    {
      "id": "audit-001",
      "actor": "ada@datastack.dev",
      "action": "deploy",
      "resourceType": "stack",
      "resourceId": "stack-001",
      "timestamp": "2024-06-10T08:00:00Z",
      "metadata": {
        "environment": "production",
        "revision": 3
      }
    }
  ],
  "totalCount": 1
}
```
