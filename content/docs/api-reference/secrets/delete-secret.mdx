---
title: Delete Secret
description: Delete a secret from a scope.
---

```
DELETE /api/v1/secrets
```

Delete a specific secret from a scope.

**Required scope:** `manage:secrets`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `scope` | `string` | Yes | Secret scope name. |
| `key` | `string` | Yes | Secret key to delete. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `message` | `string` | Confirmation message. |

## Example

```bash title="cURL"
curl -X DELETE /api/v1/secrets \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{ "scope": "aws-prod", "key": "access_key_id" }'
```

```json title="Response"
{
  "message": "Secret 'access_key_id' deleted from scope 'aws-prod'."
}
```
