`POST /api/v1/jobs/{jobId}/reset`

Reset job

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `jobId` | `string` | Yes | The jobId value. |

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `fromCheckpoint` | `string` | No | The fromCheckpoint value. |
| `clearState` | `boolean` | No | The clearState value. |

## Response `200`

Success

## Example

```bash title="cURL"
curl -X POST "/api/v1/jobs/{jobId}/reset" \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
  "fromCheckpoint": "example-fromCheckpoint",
  "clearState": true
}'
```
