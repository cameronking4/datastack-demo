---
title: Get Notification Channel
description: Get notification channel details including delivery statistics.
---

```
GET /api/v1/notifications/{notificationId}
```

Returns full details for a single notification channel.

**Required scope:** `read:alerts`

## Path Parameters

| Parameter | Type | Description |
|---|---|---|
| `notificationId` | `string` | The notification channel identifier. |

## Response `200`

| Field | Type | Description |
|---|---|---|
| `id` | `string` | Channel identifier. |
| `name` | `string` | Display name. |
| `type` | `string` | Channel type. |
| `config` | `object` | Channel-specific configuration. |
| `eventFilters` | `string[]` | Event types this channel receives. |
| `severityFilter` | `string[]` | Severity levels. |
| `enabled` | `boolean` | Whether the channel is active. |
| `quietHours` | `object` | Quiet hours configuration. |
| `createdAt` | `string` | ISO 8601 creation timestamp. |

## Example

```bash title="cURL"
curl /api/v1/notifications/notif-001 \
  -H "Authorization: Bearer <accessToken>"
```

```json title="Response"
{
  "id": "notif-001",
  "name": "Engineering Slack",
  "type": "slack",
  "config": {
    "slackChannel": "#eng-alerts"
  },
  "severityFilter": ["error", "critical"],
  "enabled": true,
  "createdAt": "2024-03-01T09:00:00Z"
}
```
