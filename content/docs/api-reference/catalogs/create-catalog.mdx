---
title: Create Catalog
description: Create a new Unity Catalog catalog.
---

```
POST /api/v1/catalogs
```

Create a new catalog in the workspace.

**Required scope:** `manage:catalogs`

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | Catalog name (must be unique). |
| `comment` | `string` | No | Catalog description. |
| `properties` | `object` | No | Key-value properties. |
| `isolation` | `string` | No | Isolation mode (`OPEN` or `ISOLATED`). Defaults to `OPEN`. |

## Response `201`

Returns the created catalog object.

| Field | Type | Description |
|---|---|---|
| `name` | `string` | Catalog name. |
| `owner` | `string` | Catalog owner. |
| `comment` | `string` | Catalog description. |
| `properties` | `object` | Key-value properties. |
| `isolation` | `string` | Isolation mode. |
| `schemasCount` | `number` | Number of schemas (starts at `0`). |
| `createdAt` | `string` | ISO 8601 creation timestamp. |
| `updatedAt` | `string` | ISO 8601 last-updated timestamp. |
| `createdBy` | `string` | Creator identifier. |

## Example

```bash title="cURL"
curl -X POST /api/v1/catalogs \
  -H "Authorization: Bearer <accessToken>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "analytics",
    "comment": "Analytics data catalog",
    "properties": { "environment": "production" }
  }'
```

```json title="Response"
{
  "name": "analytics",
  "owner": "api",
  "comment": "Analytics data catalog",
  "properties": { "environment": "production" },
  "isolation": "OPEN",
  "schemasCount": 0,
  "createdAt": "2024-06-10T10:00:00Z",
  "updatedAt": "2024-06-10T10:00:00Z",
  "createdBy": "api"
}
```
